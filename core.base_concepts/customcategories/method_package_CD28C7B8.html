Основное описание

Пакеты методов обычно применяются для объединения элементов методов в иерархические структуры.  Все элементы метода должны находиться в одном и том же пакете методов.

Предусмотрены два типа пакетов методов:

  • Пакет наполнения
  • Пакет процессов
