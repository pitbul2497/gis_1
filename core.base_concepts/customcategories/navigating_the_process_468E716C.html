Основное описание

Сайт метода  состоит из набора страниц в формате HTML, которые можно просматривать с помощью браузеров Microsoft® Internet Explorer, Mozilla и Firefox.

Примечание: для работы некоторых функций, реализованных в виде аплетов, например Поиск и Мое представление, требуется JRE 1.4.2 или одной из последующих версий (JRE можно загрузить с сайте http://java.sun.com/j2se). Кроме того, можно опубликовать конфигурацию без аплетов - в этом случае не будут доступны функции Поиск и Мое представление, но и не будет нужна JRE. Если просматриваемый сайт был опубликован без аплетов, функции  Поиск и Мое представление будут недоступны.

Для просмотра кратких инструкций по просмотру сайта щелкните на нужной области рисунка.

Элементы среды браузера

Глоссарий

Нажав эту кнопку, можно открыть отдельное окно с глоссарием, в котором в алфавитном порядке перечислены термины сайта метода и приведены их определения и ссылки на страницы с примерами.

Индекс

Нажав на кнопку Индекс, можно перейти в новое окно с алфавитным списком разделов сайта метода. Щелкнув на ссылке в оглавлении, можно открыть текст соответствующей страницы в основном окне.

Отзывы

Нажав кнопку Отзывы, можно перейти в окно отправки отзыва о Rational по электронной почте. В отзыв будет автоматически вставлена ссылка на страницу, открытую в основном окне.

О программе

При нажатии кнопки О программе открывается окно с информацией о версии программы и авторских правах.

Поиск

Функция поиска позволяет выполнять поиск разделов сайта по ключевым словам. Все страницы, тема которых имеет отношение к указанным ключевым словам, будут показаны в окне результатов поиска. Эта функция осуществляет поиск ключевых слов в описании разделов, а не полнотекстовый поиск по страницам.
Учтите, что  функция поиска реализована в виде аплета, и поэтому для работы с ней нужна JRE 1.4.2 или одной из последующих версий (JRE можно загрузить с сайта http://java.sun.com/j2se). Если просматриваемый сайт был опубликован без аплетов, функция  Поиск будет недоступна.

Представления

Представление - это альтернативный способ навигации по страницам сайта. Ниже приведены несколько примеров представлений.

  1. В представлении Коллектив показано все содержимое сайта. В него входят все прочие представления, что позволяет просматривать метод с различных точек зрения (жизненный цикл, процесс поставки, роли, дисциплины, продукты работы и т.п.). Кроме того, в этом представлении приведена информация из представления Начало работы.
  2. Представление Начало работы содержит ознакомительную информацию о методе (ключевые принципы, основные сведения о процессах, основные сведения об архитектуре методов), инструкции по навигации и путеводители по сайту для различных заинтересованных лиц и с точки зрения различных технологий. Кроме того, предусмотрены дополнительные материалы, включая ссылки и ресурсы.
  3. Представление Наборы ролей позволяет выбрать интересующую вас роль и просмотреть всю информацию о ней. Кроме того, можно просматривать дисциплины, домены и памятки по инструментам, имеющие отношение к данной роли.
  4. Представление процессы поставки предназначено для навигации по жизненным циклам (например, Classic RUP или Business Modeling). В иерархическую структуру включены элементы жизненного цикла на уровне этапов, итераций, операций и элементов операций.  Для каждого элемента жизненного цикла показаны три структуры:
    • Структура распределения работы, в которой показаны операции и задачи
    • Распределение ролей 
    • Использование продуктов работы  

Учтите, что в структуре распределения работы могут быть предусмотрены активные диаграммы, иллюстрирующие работу, связанную с соответствующим элементом жизненного цикла. Кроме того, задачи в структуре распределения работы разбиты на отдельные шаги. Эти шаги показаны в виде синих кругов. Синими окружностями обозначены шаги, которые не выполняются в данном жизненном цикле.
