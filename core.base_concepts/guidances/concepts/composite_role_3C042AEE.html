Концепция: Составная роль
Составная роль - это особый вид роли, в состав которого входят несколько ролей. Составные роли применяются для группировки ролей в целях сокращения общего количества ролей в наполнении метода для процесса.
Основное описание

Составная роль представляет собой группу ролей, которой можно пользоваться для сокращения количества ролей в операции или процессе. Таким образом, составные роли распространяются на задачи и продукты работы ролей, входящих в эти составные роли. Составные роли часто применяются с процессами, разработанными для малых рабочих групп, в которых каждый сотрудник выполняет несколько формальных ролей наполнения метода. Составные роли можно рассматривать в качестве инструмента объединения ролей в кластеры.

Например, можно создать составную роль разработчик, в которую будут входить роли реализатор и испытатель. Тогда всякий раз, когда в структуре будет указана роль реализатора или испытателя, вместо нее будет применяться роль разработчика. Если к процессу будет добавлен дескриптор задачи, для которой в качестве главного исполнителя будет указан реализатор или испытатель, эта роль будет автоматически заменена составной ролью разработчика, объединяющей роли реализатора и испытателя.
