Концепция: Пользовательская категория
Пользовательские категории применяются для распределения наполнения по категориям, удобным для пользователя. Одно из полезных применений пользовательских категорий заключается в создании представлений.
Основное описание

Пользовательская категория - это категория, созданная для создания произвольной структуры элементов наполнения метода, удовлетворяющих определенным критериям. Пользовательские категории применяются для систематизации наполнения, а также для создания иерархических структур вложенных категорий для просмотра наполнения метода и процессов, основанных на этих категориях. Эта разновидность пользовательских категорий называется представлениями.

Примеры

Можно создать пользовательскую категорию для распределения наполнения по отделам компании, разрабатывающей программное обеспечение. Например, в категорию "тестирование" можно включить все роли, продукты работы, задачи, шаблоны возможностей и указания, относящиеся к тестированию.

Другим примером может служить создание категорий, соответствующих уровням лицензирования наполнения. Таким образом можно отделить бесплатно распространяемое наполнение от наполнения, составляющего интеллектуальную собственность и передаваемого только по лицензии.
