Концепция: Набор ролей
Эта категория применяется для упорядочения и классификации ролей.
Основное описание

Наборы ролей применяются для систематизации ролей по признакам.

Например, в разработке программного обеспечения набор ролей "аналитики" может содержать следующие роли:

  • аналитик бизнес-процессов
  • системный аналитик
  • разработчик требований

Все эти роли связаны с применением близких по духу технологий, для их выполнения требуются схожие навыки, однако сами роли должны различаться при разработке программного обеспечения.
