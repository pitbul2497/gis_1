Предметный указатель
Предметный указатель позволяет осуществлять поиск разделов по ключевым словам.
Основное описание

Предметный указатель позволяет осуществлять поиск разделов по ключевым словам. Ключевые слова, из которых сформирован предметный указатель, указываются при создании страниц. В верхнем фрейме предметного указателя показан список разделов, начинающихся с определенной буквы или цифры. В нижнем фрейме показан список разделов и ссылок. Щелкнув на ссылке, можно открыть текст соответствующей страницы в основном окне.
