Операция: Изменение элементов конфигурации и организация их поставки
Эта операция применяется для управления артефактами проектов и выполнением связанных с этим операций, начиная с создания частных артефактов и заканчивая поставкой общих артефактов членам проектной группы и другим заинтересованным лицам.

Описание

Данная операция сфокусирована на решении следующих задач:

  • Создание рабочих областей, оценка артефактов проекта, изменение артефактов, публикация изменений для включения в окончательный продукт пользователями с любыми ролями в рамках проекта.
  • Компоновка продукта, создание контрольных версий и публикация контрольных версий для остальных членов группы разработчиков.

Персонал

Выражение "любая роль" в данном контексте обозначает любую роль RUP, исполнителю которой разрешено вносить изменения в артефакты (элементы конфигурации) RUP.

Интегратор (как указано в описании других операций) должен обеспечить тестирование артефактов, полученных из рабочих областей разработки, в достаточном объеме для внедрения этих артефактов в компонуемый продукт. Интегратор должен знать правила тестирования и быть знаком со стратегиями управления конфигурацией проекта.
