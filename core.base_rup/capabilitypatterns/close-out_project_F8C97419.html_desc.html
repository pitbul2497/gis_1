Описание

Готовится заключительная оценка состояния для приемки продукта, и если приемка будет пройдена успешно, формальным владельцем программного продукта становится заказчик. Руководитель проекта закрывает проект путем удаления оставшихся ресурсов и передачи освободившегося персонала в другой проект.
