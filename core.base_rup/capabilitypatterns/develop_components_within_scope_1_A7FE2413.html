Описание

Данный шаблон (набор операций) обычно повторяется для каждой процедуры компоновки в ходе итерации.  Частота компоновки зависит от конкретного проекта.   Иногда компоновка выполняется ежедневно, а иногда - один раз за всю итерацию.

Словосочетание [в пределах содержания] в имени данного шаблона значит, что в данном шаблоне предусмотрена компоновка только тех операций, которые входят в состав определенного содержания, например, в определенный компонент.   Данным шаблоном можно пользоваться многократно, по разу для каждой разновидности содержания.  Например, им можно пользоваться по разу для каждого компонента в составе итерации.
