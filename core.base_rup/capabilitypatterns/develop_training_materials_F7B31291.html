Разработайте общие характеристики учебных материалов

Определите целевую аудиторию обучения. Какие позиции занимают в своей организации типичные слушатели курса? Какими обязательными качествами должны обладать слушатели курса?

Определите задачи курса. С какими функциями продукта нужно познакомить слушателей? Во многих случаях в рамках одной сессии обучения невозможно охватить все тонкости работы с продуктом. Какой уровень понимания продукта должны вынести слушатели? Некоторые аспекты достаточно только обозначить, тогда как некоторые, необходимые функции, нужно обязательно разъяснить так, чтобы слушатели могли применять их сразу после завершения курса.

Выберите форму обучения. Например:

  • Обучение через сеть Internet.
  • Обучение в аудитории.
  • Семинары.

Определите продолжительность обучения и максимальное количество слушателей. Рекомендуется разработать не один, а несколько разных курсов, чтобы они подходили для разных аудиторий.

Проработайте техническую сторону выпуска учебных материалов. Обычно для этой задачи обращаются к специализированным компаниям.

Напишите учебные материалы

Прежде всего следует сконцентрироваться на создании тех материалов, которые слушатели будут использовать в процессе занятий. Если вы разрабатываете курс для чтения в аудитории или семинарский курс, не забудьте добавить слайды и необходимые упражнения и задачи.

Если учебные материалы будут использоваться для проведения занятий не только самим автором, составьте также рекомендации для преподавателей и инструкции по разработке материалов.

В зависимости от темы занятия рекомендуется составить список дополнительной литературы и ссылок в Internet, где можно получить более подробную информацию. В некоторых случаях можно раздать слушателям распечатанные выдержки из дополнительных источников.

Для обеспечения качества учебных материалов автор может собрать консультативный совет для их просмотра и обсуждения отдельных моментов. В этот совет можно пригласить пользователей, специалистов по сбыту и разработчиков продукта.
