Операция: Управление заключительным тестированием при индивидуальной установке
Данная операция заключается в проверке пригодности продукта к эксплуатации перед тем, как передать его заказчику. Это особая разновидность заключительного тестирования для случаев, когда предполагается индивидуальная установка продукта в среде заказчика.
