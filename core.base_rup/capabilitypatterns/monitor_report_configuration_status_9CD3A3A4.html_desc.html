Описание

Данная операция сфокусирована на решении следующих задач:

  • Обеспечение доступности артефактов и их контрольных версий.
  • Определение наличия необходимых артефактов в управляемой библиотеке и наличия контрольных версий этих артефактов.
  • Поддержка операций, связанных с учетом состояния конфигурации проекта.
  • Организация создания отчетов о запросах изменений, в первую очередь для операций, связанных с исправлением ошибок и реализацией новых функций.
  • Организация сбора данных и создания отчетов для контроля хода разработки и отслеживания тенденций.

Указания по использованию

Эта процедура выполняется на всем протяжении жизненного цикла продукта, начиная с начального этапа.

Эта процедура считается обязательной, хотя ее объем сильно зависит от конкретного проекта. Как правило, сложность этой процедуры прямо пропорциональна численности участников проектной группы.
