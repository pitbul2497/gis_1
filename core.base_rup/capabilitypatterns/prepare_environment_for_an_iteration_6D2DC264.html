Назначение
Цель этой операции заключается в подготовке среды к выполнению итерации. Среда состоит из процесса и инструментов .

Описание

В ходе каждой итерации данная операция  сфокусирована в основном на решении следующих задач:

  • Подготовка прецедента разработки для итерации.
  • Подготовка и, при необходимости, настройка инструментов, применяемых в итерации.
  • Проверка правильности настройки и установки инструментов.
  • Подготовка набора шаблонов и рекомендаций для поддержки разработки артефактов проекта в ходе итерации.
  • Подготовка механизма уведомления всех членов проектной группы о любых изменениях в среде проекта.

Персонал

Разработчик процессов отвечает за подготовку прецедента разработки. Специалисты, владеющие сразу всеми аспектами разработки программного обеспечения, встречаются довольно редко, и поэтому разработчик процесс должен консультироваться с экспертами, как внутри отдела разработки, так и за его пределами.

В разработке рекомендаций и шаблонов для проекта могут участвовать несколько специалистов. Для этого не требуется подробное знание процесса.

Обычно роль специалистов по инструментам выполняют несколько сотрудников. Каждый из них отвечает за отдельный инструмент или группу инструментов.

Использование
Указания по использованиюКак видно из имени этой операции, она выполняется в начале каждой итерации.  
Ключевые условия

Рекомендуется выполнять эту процедуру во всех проектах, хотя иногда некоторые из перечисленных выше задач выполняются неформально и рассматриваются как составная часть текущей поддержки среды проекта. Дополнительные сведения о текущей поддержке приведены в разделе Операция: поддержка среды во время итерации.
