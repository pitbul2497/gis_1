Операция: Подготовка среды для проекта
Данная операция заключается в подготовке среды разработки проекта, включая процессы и инструменты.

Назначение
Цель этой операции заключается в адаптации процесса разработки  и подготовке инфраструктуры разработки для конкретного проекта.

Описание

Подготовка среды разработки заключается в выполнении следующих операций:

  • Определение того, как в проекте будет использоваться настроенный процесс разработки.
  • Проверка выбора продуктов работы и требований к продолжительности разработки и формальной стороне процесса.
  • Подготовка уникальных ресурсов для проекта, например рекомендаций и шаблонов.
  • Подготовка ориентировочного списка инструментов для разработки продукта.

Персонал

Рекомендуется поручить выполнение этой процедуры небольшому коллективу сотрудников, хорошо знакомых с общим процессом разработки. Если в организации есть специальный отдел, занимающийся вопросами процесса и среды разработки, роль Роль: разработчик процесса обычно назначается сотруднику этого отдела.

Использование
Указания по использованию

Подготовка среды проекта выполняется на начальном этапе. Все последующие действия по оптимизации и коррекции подготовленной среды выполняются в рамках процедуры Операция: подготовка среды для итерации.

Ключевые условия

Рекомендуется уделить достаточное внимание подготовке среды разработки с учетом потребностей проекта.

Информация об уровне организации разработки обычно собирается до начала проекта и документируется в объекте Продукт работы: оценка организации разработки.
