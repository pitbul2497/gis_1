Описание

В ходе этой операции создается блок развертывания.   В состав блока развертывания входят все программное обеспечение и сопутствующие продукты, необходимые для его установки и эксплуатации.   Конечный продукт  состоит из блока развертывания, артефактов установки и документации по продукту.

Модули развертывания создаются для тестирования бета-версий и конечного продукта.

Для выполнения данной операции требуются навыки, необходимые для создания продукта,   сценариев установки и справочной документации в том формате, в котором эти материалы наиболее удобно предоставлять конечным пользователям.

Указания по использованию

Данная процедура в основном выполняется на этапах построения и внедрения, хотя может начаться и на начальном этапе, особенно если отдельные аспекты развертывания продукта связаны с серьезными архитектурными рисками. Обычно эта процедура начинается в ходе выполнения итерации и до завершения итерации может быть выполнена несколько раз.

Данная процедура считается обязательной, хотя ее формат, стиль и объем сильно зависят от конкретного проекта.
