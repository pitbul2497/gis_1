Планирование собрания, посвященного утверждению проекта

В первую очередь следует определить участников процесса утверждения проекта и назначить время собрания для обсуждения предложенного проекта. В большинстве организаций такие вопросы решает специальная административная комиссия. Как правило, в состав этой комиссии входят представители различных областей бизнеса.

После определения участников собрания следует запланировать дату и время его проведения. Важно выделить участникам достаточное время для проверки материалов, на основе которых будет приниматься решение об утверждении.

Распределение материалов собрания

Перед собранием участникам следует раздать материалы проекта. Рекомендуется заблаговременно отправлять эту информацию, оставив достаточное время на знакомство с ней. Минимальный набор рабочих продуктов, подлежащих проверке, выглядит следующим образом:

  • Видение
  • Экономическое обоснование проекта
  • Список рисков
Проведение собрания, посвященного утверждению проекта

В ходе собрания участники оценивают достоинства предлагаемого проекта. В результате должно быть принято согласованное решение о дальнейшей судьбе проекта. Рекомендуется обратить внимание на следующие вопросы:

  • Преимущества, предоставляемые заказчику
  • Преимущества, предоставляемые внутреннему бизнесу
  • Технические преимущества
  • Приемлемы ли расходы на проект?
  • Насколько эффективен возврат инвестиций (ROI)?
  • Требуемые ресурсы доступны или достижимы?
  • Достижимы ли требования проекта на начальном этапе?

В конце собрания участники должны принять решение относительно утверждения проекта. Возможны следующие результаты:

Утвержден Проект будет продолжен. Выделяются средства и ресурсы для выполнения по крайней мере начального этапа.  
Отклонен  Проект недостаточно привлекателен для продолжения.
Решение отложено Для принятия окончательного решения требуется более подробная информация или дополнительный анализ.  
Запись решения

По завершении собрания создается протокол, включающий важные обсуждения, требующие внимания вопросы и полученные результаты. Если решение было отложено, предварительно планируется повторное собрание.
