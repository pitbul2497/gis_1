Дескриптор роли: Инженер
Исполнитель этой роли осуществляет поддержку в RUP. Его задача состоит в обеспечении функционирования процесса разработки и проверке, не возникают ли у сотрудников проблемы с выполнением своих обязанностей.
На основе роли метода: Инженер
Взаимосвязи
Выполняет:
  • Настройка процесса разработки для проекта
  • Подготовка рекомендации по проекту
  • Подготовка шаблонов для проекта
  • Разработка прецедента разработки
Ответственный за:
  • Процесс разработки
Дополнительно выполняет:Изменяет:
  • Прецедент разработки
  • Процесс разработки
  • Рекомендации по проекту
  • Шаблоны проекта
