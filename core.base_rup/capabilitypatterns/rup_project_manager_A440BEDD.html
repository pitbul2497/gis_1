Дескриптор роли: Руководитель проекта
Исполнитель этой роли состоит планирует затраты ресурсов, распределяет ресурсы, выделяет приоритеты, управляет связями с потребителями и пользователями и направляет усилия коллектива. Руководитель проекта также должен обеспечивать целостность и качество рабочих продуктов проекта.
На основе роли метода: Руководитель проекта
Взаимосвязи
Выполняет:
  • Начало итерации
  • Определение и оценка рисков
  • Оценка итерации
  • Получение персонала
Ответственный за:
  • Оценка итерации
  • Оценка состояния
  • План итерации
  • План разработки программного обеспечения
  • Порядок обработки
  • Список дефектов
  • Список рисков
  • Экономическое обоснование проекта
Дополнительно выполняет:Изменяет:
  • Оценка итерации
  • План разработки программного обеспечения
  • Порядок обработки
  • Список рисков
