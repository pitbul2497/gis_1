Дескриптор роли: Разработчик архитектуры
Задача исполнителя этой роли состоит в управлении разработкой программной архитектуры системы, что включает продвижение и поддержку ключевых технических решений, задающих рамки модели и реализации проекта.
На основе роли метода: Разработчик архитектуры
Взаимосвязи
Выполняет:
  • Объединение существующих структурных элементов
  • Определение механизмов проектирования
  • Определение элементов проекта
  • Создание описания архитектуры среды выполнения
  • Создание описания варианта
  • Создание структуры модели реализации
Ответственный за:
  • Документ архитектуры программного обеспечения
  • Модель анализа
  • Модель проекта
  • Модель развертывания
  • Модель реализации
Дополнительно выполняет:
  • Анализ операции
Изменяет:
  • Документ архитектуры программного обеспечения
  • Модель проекта
  • Модель развертывания
  • Модель реализации
