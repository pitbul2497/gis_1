Дескриптор роли: Аналитик
Исполнитель этой роли определяет необходимые тесты, отслеживает выполнение тестирования и его результаты и оценивает качество продукта в целом. Эта роль также представляет заинтересованных лиц, не участвующих непосредственно в проекте.
На основе роли метода: Аналитик
Взаимосвязи
Выполняет:
  • Определение идеи тестирования
  • Определение потребностей в оценке и прослеживаемости
  • Определение сведений о тесте
Ответственный за:
  • Данные теста
  • Модель анализа нагрузки
  • Тестовый набор
  • Тест - Список идей
Дополнительно выполняет:Изменяет:
  • Данные теста
  • Модель анализа нагрузки
  • План тестирования
  • Сценарий тестирования
  • Тестовый набор
  • Тест - Список идей
