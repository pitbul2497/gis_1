Описание

Для структурирования процедур, входящих в состав дисциплины тестирования, совокупность операций и продуктов работы дисциплины называется шаблоном ее возможностей.

Каждая операция предназначена для решения некоторой общей задачи, без которой невозможно эффективное тестирование продукта.

Точный состав данного шаблона возможностей зависит от потребностей конкретных итераций и проектов.

Выберите способ выполнения потока операций



Необходимо принять несколько решений в отношении потока операций дисциплины тестирования:

  • Проанализируйте поток операций и выберите способ его выполнения. Изучите диаграмму и указанные на ней сторожевые условия с учетом приведенных ниже рекомендаций.
  • Решите, какие операции тестирования нужно выполнить. Одна из самых сложных задач в ходе тестирования заключается в определении того, какие рамки качества нужно установить для проекта в целом и, что еще более важно, для каждой отдельной итерации (см. Концепция: типы тестов). Выберите сочетания типов тестов, на которых вы хотите акцентировать внимание в текущей итерации.
