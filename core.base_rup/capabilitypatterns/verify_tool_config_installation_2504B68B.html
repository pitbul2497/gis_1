Шаги
Проверка среды

Убедитесь, что среда разработки содержит подходящее аппаратное и программное обеспечение, а также правильные данные. Убедитесь, что требуемое аппаратное обеспечение установлено. Это можно сделать визуально или с помощью инструментов (например, контекстное меню Свойства значка Мой компьютер в операционной системе Windows).  

Проверка инструментов

Убедитесь, что задана правильная конфигурация программного обеспечения. Для этого можно просмотреть параметры реестра, файлы 'ini' или вывод соответствующих инструментов.

Проверка данных

Проверьте правильность данных. Проверка данных может потребовать применения инструментов для визуального осмотра или приложений просмотра данных. На некотором этапе для каждого инструмента можно выбрать один или несколько вариантов использования (сценариев) и убедиться, что результаты использования инструмента согласованы с потребностями.

Запуск инструментов

Соберите небольшую группу сотрудников, которые хорошо разбираются в инструментах и процессе разработке, и предложите им поработать с инструментами.  

  • Выполните тестирование из нескольких расположений в условиях одновременной работы большого числа пользователей.
  • Для каждого инструмента выполните по крайней мере один сценарий варианта использования, проверив тем самым правильность установки и настройки.  
  • Проверьте стандартные сценарии процесса разработки и рекомендаций по инструментам.   
  • Проверьте интеграцию между различными инструментами.  

При необходимости отправьте запросы изменений.
