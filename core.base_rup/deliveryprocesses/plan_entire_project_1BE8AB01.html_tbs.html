Операция: Создание планов проекта
Данная операция заключается в создании общего плана выпуска. Она не охватывает подробное планирование тестирования проекта (эта процедура входит в состав процедуры оценки качества).

Данная операция охватывает общее планирование выпуска. Сюда не входит подробное планирование тестирования продукта.
