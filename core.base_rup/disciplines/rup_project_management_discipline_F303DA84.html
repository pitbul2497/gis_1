Основное описание

В этом разделе приведена общая информация об управлении проектами. Это не готовый рецепт успеха, а скорее описание подхода к управлению проектами, который позволяет значительно повысить вероятность разработки хорошего продукта.

Цель управления проектом  заключается в следующем:

  • Создать среду для управления проектами, в которых активно используется программное обеспечение.
  • Предоставить практические рекомендации по планированию проекта, подбору персонала, выполнению проекта и контролю хода его выполнения.
  • Создать среду управления рисками.

Перед этой дисциплиной Rational Unified Process (RUP) не ставилась задача охватить все аспекты управления проектами. В частности, в нее не входят следующие вопросы:

  • Управление персоналом: подбор, обучение и инструктирование
  • Управление бюджетом: формирование, распределение и т.п.
  • Управление контрактами с клиентами и поставщиками.

Эта дисциплина в первую очередь посвящена важнейшим элементам итерационного процесса разработки:

  • Управление рисками
  • Планирование итерационного процесса как на уровне жизненного цикла, так и на уровне отдельных итераций
  • Контроль хода выполнения итерационного проекта и оценка ключевых показателей

Дисциплина управления проектами образует среду для создания проектов и управления ими. При этом в работе над проектом задействуются все остальные дисциплины:

  • Управление требованиями
  • Анализ и проектирование
  • Реализация
  • Тестирование
  • Развертывание

Управление проектом входит в группу дисциплин поддержки процесса наряду со следующими дисциплинами:

  • Управление конфигурацией и изменениями
  • Управление средой

Дополнительные источники информации по дисциплине управления проектами института Project Management Institute

Перед разработчиками RUP не стояла задача создания полного учебного курса по руководству проектами. Здесь описаны только те элементы этой дисциплины, которые напрямую связаны с обсуждаемым подходом к разработке программного обеспечения, причем даже не все такие элементы. Некоторые темы, как указано выше, затрагиваются только поверхностно или не рассматриваются вовсе. Подход к управлению проектами, описанный в данном материале, был разработан по материалам руководства руководства по управлению проектами института PMI (PMBOK®). Полное описание рекомендуемого подхода к управлению проектами можно найти в этом руководстве. Его можно загрузить с сайта www.pmi.org. Институт PMI® (Project Management Institute) - ведущая некоммерческая организация, занимающаяся вопросами управления проектами. Она разрабатывает стандарты управления проектами и проводит профессиональную сертификацию.
