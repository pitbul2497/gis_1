Справочная таблица: Модель данных
Данная справочная таблица применяется для проверки правильности и полноты модели данных.
Взаимосвязи
Связанные элементы
  • Модель данных
  • Проектирование базы данных
Основное описание
Элементы справочной таблицы
Все постоянные классы, пользующиеся базой данных для обеспечения постоянства, сопоставлены структурам базы данных
Создана таблица пересечений для отношений "многие-ко-многим"
Для всех таблиц указаны первичные ключи, кроме случаев, когда это намеренно не сделано для повышения производительности
Оптимизированы операции сохранения и извлечения данных
Если применяется реляционная база данных, таблицы денормализованы для повышения производительности.
Если применяется денормализация, установлено, что она не снижает производительность обновления, вставки и удаления данных ни в одном из сценариев.
Для ускорения доступа созданы индексы
Необходимость обновления индекса учтено в других операциях над таблицей
Запланировано распределение данных
Определены ограничения по ссылкам и данным
Разработан план контроля ограничений по ссылкам при изменении правил для данных
Определены хранимые процедуры и триггеры
В механизме обслуживания постоянных классов реализованы единообразные правила работы с хранимыми процедурами и триггерами базы данных
