Справочная таблица: Модель проектирования
Данная справочная таблица позволяет проверить правильность определения модели проекта.
Взаимосвязи
Связанные элементы
  • Модель проектирования
Основное описание
Элементы справочной таблицы
Общие
  • Четко определены и видимы цели модели.
  • Подробность модели соответствует установленным в ней целям.
  • Выбор конструктивных элементов модели соответствует характеру решаемой задачи.
  • Модель максимально проста, но при этом обеспечивает решение поставленных задач.
  • Модель достаточно гибка для внесения ожидаемых изменений в будущем.
  • Проект соответствует решаемой задаче (не слишком сложный и не слишком громоздкий).
  • Проект прост для понимания и обслуживания.
  • Реализация проекта реалистична.
Уровни
  • Используется не больше семь (плюс-минус 2) уровней.
  • Существует четкое обоснование системы уровней и реальная структура уровней полностью соответствует ему.
  • В пределах проекта соблюдаются границы уровней.
  • Уровни применяются для инкапсуляции границ между различными видами услуг и формируют удобные абстракции, упрощающие понимание проекта.
