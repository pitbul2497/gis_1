Основное описание
Элементы справочной таблицы
Всем пакетам присвоены уникальные имена, отражающие их назначение.
Описания пакетов точно передают их общее назначение.
Экспортируемые классы пакета образуют единую и целостную совокупность служб.
Зависимости между пакетами соответствуют зависимостям между содержащимися в них классами
Содержимое пакета представляет собой совокупность тесно связанных элементов, которые слабо связаны с элементами других пакетов.
Дальнейшее разделение пакета на несколько сравнительно независимых наборов, в пределах каждого из которых элементы тесно связаны друг с другом, невозможно.
Общее количество пакетов пропорционально общему количеству элементов моделирования.
