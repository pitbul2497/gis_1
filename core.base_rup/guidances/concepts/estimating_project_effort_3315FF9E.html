Введение

Одной из самых востребованных величин в сфере разработки программного обеспечения является точный предварительный прогноз общей стоимости проекта. Учитывая всеобщую тенденцию к нарушению графиков и невыполнению бюджетов разработки, это не удивительно. Однако в настоящее время разработку программного обеспечения нельзя назвать точной наукой, о чем недвусмысленно свидетельствует огромное количество провалившихся проектов.

Тем не менее, не помешает хотя бы попытаться спрогнозировать общую стоимость проекта, а затем сверять прогноз с фактическими данными в ходе выполнения проекта. Ниже перечислены некоторые технологии, применяемые для прогнозирования стоимости проектов.

Показатели вариантов использования

Данная технология основана на результатах работы, проделанной в компании Objective Systems SF AB, и была впервые опубликована в 1993 году в статье Густава Карнера Resource Estimation for Objectory Projects. Эта технология позволяла оценить объем ресурсов, которые потребуются для разработки программной системы, на базе элементов модели вариантов использования. Метод позволяет вычислить так называемые показатели вариантов использования (UCP), которые затем преобразуются в функциональные показатели (FP). Функциональные показатели (FP) представляют собой довольно распространенную систему прогнозирования, разработанную А. Дж. Альбрехтом (1979), см. Measuring application development productivity.

Дополнительные сведения можно найти в разделе Информационный бюллетень: Оценка трудоемкости на основе вариантов использования.

Wideband Delphi

Эта технология, разработанная компанией Rand Corporation в 1948 году, первоначально называлась методом оценки Delphi. Суть метода заключается в том, что создается группа экспертов, каждый из которых дает анонимную оценку исходя из описания задачи, а затем эксперты итерационным путем сводят это множество оценок к одной.

В начале 1970-х годов Барри Бем (Barry Boehm) и его коллеги из корпорации Rand расширили базовый метод путем усиления взаимодействия групп экспертов, и в результате увидел свет метод Wideband Delphi (см. Software Engineering Economics. [BOE81]. Позднее Мэри Сакри и Нил Поттер из консультационной компании The Process Group в Далласе (Техас) разработали устойчивую процедуру проведения оценки по методу Wideband Delphi для проектов по разработке программного обеспечения. В данном руководстве излагается эта процедура.
