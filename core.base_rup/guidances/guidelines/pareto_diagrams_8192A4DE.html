Рекомендация: Диаграммы Парето
Диаграммы Парето используются совместно с диаграммами причинно-следственных связей для иллюстрации веса причин неполадки. Далее приводится пример создания диаграммы Парето.
Взаимосвязи
Связанные элементы
  • Определение требований заинтересованных лиц
  • Разработка видения
  • Создание подробного описания требований к программному обеспечению
Основное описание

Диаграммы Парето можно использовать совместно с диаграммами причинно-следственных связей (Фишбона). Первые концентрируются на относительном весе причин неполадки.

Определите процентное соотношение веса причин неполадки. Нарисуйте диаграмму Парето и примите решение на основе 20 самых весомых причин.
