Работа с расширенной справкой
Расширенная справка позволяет просматривать инструкции IBM Rational Unified Process(R) по разделам, относящимся к применяемым вами инструментам Rational.
Основное описание

Обзор

Расширенная справка позволяет просматривать инструкции IBM Rational Unified Process по разделам, относящимся к применяемым вами инструментам Rational. Например, можно запустить Расширенную справку из Rational Rose, не прерывая поток операций.

Для работы с Расширенной справкой необходимо установить RUP(R). Расширенная справка становится доступной сразу после завершения установки.

При выборе Расширенной справки из другого меню Справка инструмента Rational разделы Расширенной справки появляются в панели деревьев возле содержимого RUP. Определенная панель деревьев идентифицируется с помощью заголовка "Расширенная справка" и имени инструмента. В панели деревьев Расширенной справки перечисляются разделы, которые применимы к используемому инструменту и содержимому инструмента.

В оставшейся части данного документа объясняется, как запустить Расширенную справку и перемещаться по ней.

Работа с расширенной справкой

Для работы с расширенной справкой выполните следующие действия.

В инструменте Rational (например, Rational Rose) в строке меню нажмите Справка> Расширенная справка.

Если имеется несколько установленных и опубликованных Web-сайтов RUP, появится диалоговое окно Выбор расположения расширенной справки с приглашением выбрать расположение, из которого следует просмотреть разделы расширенной справки. Для каждого Web-сайта RUP существует расширенная справка, и каждый Web-сайт может содержать различные разделы, относящиеся к используемому инструменту.

В списке в верхней части диалогового окна выберите необходимое расположение. При желании можно отметить опцию (Сделать выбранное расположение расположением по умолчанию), что сделает данное расположение расположением по умолчанию при следующем просмотре расширенной справки. Еще одна опция (Больше не показывать это окно) позволяет больше никогда не просматривать это диалоговое окно. Если эта опция выбрана, расположение расширенной справки по умолчанию будет впоследствии запускаться автоматически.

Откроется Web-браузер, показывающий дерево расширенной справки, в котором перечислены соответствующие разделы. Первая страница представляет собой обзор, предоставляющий базовую информацию о расширенной справке. Например, при запуске расширенной справки из Rational Rose дерево идентифицируется вкладкой Расширенная справка - Rose.

Сначала перечисляются 15 наиболее подходящих разделов, а затем - остальные разделы в папке Дополнительно.

Для просмотра раздела перейдите к этому разделу и дважды щелкните на нем, чтобы развернуть его в браузере. Дерево расширенной справки останется слева, а содержимое раздела появится справа.

Для просмотра разделов в папке Дополнительно дважды щелкните на ней, чтобы развернуть папку в дереве.

Если необходимо просмотреть разделы из других расположений расширенной справки щелкните на последнем узле в дереве, который называется Выбрать другое расположение расширенной справки. Снова откроется диалоговое окно Выбор расположения расширенной справки, и можно будет выбрать другое расположение для запуска другой расширенной справки. Откроется Web-браузер, показывающий соответствующую панель расширенной справки.
