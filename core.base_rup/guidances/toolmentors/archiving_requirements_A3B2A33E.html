Руководство по инструменту: Архивирование требований с помощью Rational RequisitePro
В этой памятке по инструменту описано архивирование требований с помощью команд архивирования Rational RequisitePro.
Инструмент: Rational RequisitePro
Взаимосвязи
Связанные элементы
  • Настройка среды управления конфигурацией (CM)
  • Разработка справочных материалов
Основное описание

Обзор

Необходимо сохранять требования проекта на определенных вехах для создания резервных копий. RequisitePro позволяет создавать моментальные копии проектов требований и хранить из в Rational ClearCase® либо просто в виде архивов RequisitePro, сохраняя копию продуктов работы проекта в архивном каталоге.

Для архивирования проектов RequisitePro можно использовать различные инструменты управления кофигурацией, такие как Intersolv PVCS® или Microsoft® Visual SourceSafe. Если ваш проект использует базу данных Microsoft® Access, вначале выполните команду RequisitePro Archive для создания файла архива, затем заархивируйте этот файл с помощью инструмента управления конфигурацией. Не забудьте добавить в архив документацию проекта. При архивации проекта, использующего межпроектную функцию трассировки, не забудьте добавить в архив все связанные проекты, чтобы заархивированный проект был полным.

Проект RequisitePro состоит из проекта требований Rational и набора документов, содержащих требования. Требования можно создать непосредственно на панели проекта или в документе Microsoft® Word в RequisitePro. Все документы, содержащие требования, указаны в соответствующе проекте RequisitePro, обслуживающем эти требования. Архив проекта RequisitePro содержит проект требований Rational и все документы, содержащие требования.

Примечание: Информация о создании контрольной версии проекта RequisitePro приведена в памятке по инструменту Контрольные версии проекта Rational RequisitePro.

Этапы работы с инструментами

Для архивирования проекта с помощью RequisitePro выберите одну из следующих опций:

  1. Архивировать проект RequisitePro в виде архива RequisitePro
  2. Архивировать проект RequisitePro в ClearCase
  3. Архивировать проект RequisitePro с помощью базы данных предприятия

1. Архивировать проект RequisitePro в виде архива RequisitePro

При архивации проекта RequisitePro в виде архива RequisitePro вы просто создаете копию всех файлов проекта в заданном пользователем каталоге. С помощью этой опции можно архивировать проекты RequisitePro с помощью систем управления конфигурациями, напрямую не поддерживаемых в RequisitePro.

Для архивации проекта RequisitePro в виде архива RequisitePro выполните следующие действия:

  1. Выберите Файл > Администрирование проекта > Архивировать > Архив RequisitePro. Откроется диалоговое окно Архивировать проект.
  2. Выберите каталог, в котором нужно сохранить резервную копию. Нажмите кнопку Обзор чтобы открыть окно выбора папки. Перейдите в нужный каталог, выберите папку и нажмите кнопку OK для возврата к диалоговому окну архивации проекта.
  3. Для добавления нового подкаталога в существующий путь щелкните в конце поля Каталог, добавьте обратную косую черту (\) и впишите имя нового подкаталога.
  4. Для применения номера ревизии к документам требований выберите переключатель РАспространить на все документы.
    Примечание: Если защита включена необходимо наличие доступа к структуре проекта для распространения номера ревизии. Также необходимо эксклюзивно открыть проект.
  5. Необязательные шаги: задайте новый номер ревизии для проекта и укажите имя заархивированной версии в поле Метка версии и описание в поле Изменить описание.
  6. Нажмите OK.

Чтобы обратиться к этому архиву в будущем просто добавьте заархивированный проект в список проектов RequisitePro:

  1. Выберите Файл > Открыть проект. Откроется диалоговое окно Открыть проект.
  2. Нажмите кнопку Добавить. Выберите заархивированный проект в указанном каталоге архивов.
  3. Нажмите Открыть и OK. Теперь заархивированный проект станет доступным в RequisitePro.

Дополнительная информация

Обратитесь к разделу Архивирование проектов с помощью команды RequisitePro Archive (Индекс: архивирование>с помощью команды Archive) электронной справки RequisitePro.

2. Архивация проекта RequisitePro в ClearCase

Для архивации проекта Rational RequisitePro в Rational ClearCase выполните следующие действия:

  1. Выберите Инструменты > Опции.
  2. В поле Панель ClearCase укажите расположение целевой Versioned Object Base (VOB) (диск, имя панели, имя vob, каталог). Путь должен начинаться с буквы диска. Для используемых для совместной работы дисков используется соглашение об именах. Это действие задает панель, в которую будут скопированы документы и база данных RequisitePro при архивации проекта.
  3. Нажмите OK для сохранения записи и закрытия окна Опции.
  4. Выберите Файл > Администрирование проекта > Архивировать > Rational ClearCase. Откроется окно архивации с помощью ClearCase.
  5. Выберите каталог, в котором нужно сохранить резервную копию. Нажмите кнопку Обзор чтобы открыть окно выбора папки. Перейдите в нужный каталог, выберите папку и нажмите кнопку OK. Вы вернетесь к окну Архивация с помощью ClearCase.
  6. Для добавления нового подкаталога в существующий путь щелкните в конце поля Каталог, добавьте обратную косую черту (\) и впишите имя нового подкаталога.
  7. Следующие шаги являются необязательными:
    • Укажите новый номер ревизии для проекта.
    • Примените номер ревизии к документам требований, выбрав переключатель Распространить на все документы. Примечание: Если защита включена необходимо наличие доступа к структуре проекта для распространения номера ревизии. Также необходимо эксклюзивно открыть проект.
    • Укажите имя заархивированной версии в текстовом поле Метка версии и введите описание в поле Изменить описание.

Дополнительная информация

Обратитесь к разделу Архивация проекта с помощью Rational ClearCase (Индекс: архивация>с помощью ClearCase) электронной справки RequisitePro.

3. Архивация проекта RequisitePro с помощью базы данных предприятия

Подробные инструкции по архивации проектов RequisitePro, использующих базу данных Oracle или Microsoft SQL Server приведены в разделе Архивация проекта базы данных предприятия (Индекс: архивация>баз данных предприятия) электронной справки RequisitePro.
