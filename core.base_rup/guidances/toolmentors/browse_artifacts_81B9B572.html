Основное описание

Обзор

В этой памятке по инструменту описывается навигация по Rational ProjectConsole с помощью установленных простых продуктов работы. ProjectConsole автоматизирует создание продуктов работы проекта, что позволяет создавать их быстро и точно.

В этом разделе содержатся ссылки на дополнительную связанную информацию.

ProjectConsole использует технологию аплетов Java для предоставления вам знакомых множественных форм навигации.  При щелчке непосредственно на тексте узла управления структурой будет показана страница продукта работы, связанная с узлом.  Если развернуть узел управления структурой, щелкнув на значке плюса, будут показаны гиперссылки, отображаемые при щелчке на тексте узла, в виде дочерних узлов, которые можно использовать для дальнейшей навигации.

Эта памятка по инструменту применима при работе с Microsoft Internet Explorer (начиная с версии 5.5)  или Netscape (начиная с версии 7.0).

Этапы работы с инструментами

Для обзора продуктов работы проекта с помощью ProjectConsole выполните следующие действия:

  1. Запустите браузер.
  2. Введите URL установочного сайта ProjectConsole. Получить URL можно у администратора ProjectConsole. Появится окно входа в систему ProjectConsole.
  3. Введите допустимое ИД пользователя и пароль. Откроется обозреватель продуктов работы ProjectConsole.
  4. Разверните следующие узлы: ProjectConsole, Проекты ClassicsCD.com, Точки продаж, Функциональные команды, Управление, Версия 1.0, Промежуточная информация, Отчеты о показателях, Все опубликованные показатели, Функциональные команды.
  5. Щелкните на тексте любого узла в разделе Функциональные команды для доступа к показателям для команды.
  6. Дополнительные продукты работы можно найти развернув следующие узлы: Проекты ClassicsCD.com, Webshop, Версия 1.0, Дисциплины RUP, Управление преоктами, Промежуточная информация, Все опубликованные показатели, Версия 1.0. Разверните узлы фазы для доступа к показателям для данной фазы.
  7. Перейдите к показателям ProjectConsole, щелкнув на значке Сводная панель на панели инструментов Rational ProjectConsole.

Примечание: Отчеты, показываемые ProjectConsole являются простыми шаблонами, созданными компоновщиком шаблонов Rational ProjectConsole, в сводной панели ProjectConsole создаются опубликованные диаграммы.
