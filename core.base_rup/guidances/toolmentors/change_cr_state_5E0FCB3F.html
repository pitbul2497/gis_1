Руководство по инструменту: Изменение состояния запроса на изменений с помощью Rational ClearQuest
В этой памятке по инструменту описывается процесс изменения состояния запроса на изменения с помощью Rational ClearQuest.
Инструмент: Rational ClearQuest
Взаимосвязи
Связанные элементы
  • Обновите запрос изменения
Основное описание

Обзор

Запросы на изменение перемещаются по шаблону, по жизненному циклу, от отправки до обработки. В ClearQuest каждая стадия этого жизненного цикла называется состоянием, а каждое перемещение из одного состояния в другое называется переходом состояния.

Администратор ClearQuest задает возможные состояния существования записей. Например, запись изменения находится в состоянии Отправка при первом входе в базу данных. Далее она может проследовать в состояние Открыто на время изучения запроса на изменение, и затем перейти в состояние Зафиксировано при исправлении дефекта.

Этапы работы с инструментами

Для изменения состояния запроса на изменений в Rational ClearQuest выполняются следующие действия:

  1. Запрос к базе данных
  2. Изменение состояния


Выберите пункт Дополнительная информация для просмотра дополнительных сведений.

1. Запрос к базе данных

ClearQuest содержит несколько запросов по умолчанию, хранящихся в папках. Для отправки запроса к базе данных выполните следующие действия:

  1. Дважды щелкните на запросе.
    При наличии в базе данных запросов на изменение, совпадающих с параметрами запроса, ClearQuest отображает их на вкладке Набор результатов.
  2. При необходимости отсортируйте запросы с помощью двойного щелчка на заголовках столбцов на вкладке Набор результатов.
  3. На вкладке Набор результатов выберите изменяемый запрос на изменение.

2. Измените состояние

  1. Нажмите кнопку Действия и выберите состояние из списка.
    Меню Действия является динамическим. Оно отображает только действия, допустимые для текущего состояния. Допустимые действия задает администратор ClearQuest. Например, в приложении по умолчанию запись Обработанные нельзя переместить обратно в состояние Открыто без проверки или отказа от исправления.
  2. Переместитесь между вкладками и заполните обязательные поля, выделенные красным.
    Инструкции по заполнению полей можно просмотреть щелкнув правой кнопкой мыши на поле и выбрав в контекстном меню пункт Сообщение о ошибке.
  3. Нажмите кнопку Применить. Для удаления изменений без сохранения нажмите кнопку Восстановить.

Дополнительная информация

Дополнительная информация приведена в электронной справке ClearQuest: выберите Справка > Содержание и индекс > Работа с записями > Изменение состояния записей.
