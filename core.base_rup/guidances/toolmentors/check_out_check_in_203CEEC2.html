Руководство по инструменту: Добавление и удаление элементов конфигурации с помощью Rational ClearCase

Основное описание

Обзор

В этой памятке по инструменту описывается добавление и изъятие с помощью базовой модели ClearCase при работе в Microsoft Windows.

Этапы работы с инструментом

Для добавления и изъятия элемента конфигурации с помощью ClearCase выполните следующие действия:

  1. Отметьте файлы, которые следует изменить
  2. Внесите изменения и выполните полнофункциональное тестирование для проверки изменений
  3. Добавление изменений

1. Отметьте файлы для изменения

  • Перейдите к обозревателю ClearCase. На панели задач Windows выберите кнопку Пуск > Программы > Rational Software > Rational ClearCase > ClearCase Explorer.
  • В ClearCase Explorer щелкните правой кнопкой на элементе или элементах для их изъятия и выберите пункт Изъять во всплывающем меню ClearCase.
  • Введите комментарий в диалоговом окне Изъять и нажмите кнопку OK.

  См. руководство по ClearCase Разработка ПО или электронную справку ClearCase для получения сведений об изъятии файлов.

2. Внесите изменения и выполните полнофункциональное тестирование для проверки изменений

Проверьте внесенные изменения в частную копию помеченных файлов или каталогов. Пометив работу вы делаете ее доступной для прочих участников коллектива.

3. Добавление изменений

  • Перейдите к обозревателю ClearCase.  
  • В ClearCase Explorer щелкните правой кнопкой на элементе или элементах для их добавления и выберите пункт Добавить во всплывающем меню ClearCase.
  • Введите комментарий в диалоговом окне Добавить и нажмите кнопку OK.

  См. руководство по ClearCase Разработка ПО или    электронную справку ClearCase для получения сведений о добавлении файлов.
