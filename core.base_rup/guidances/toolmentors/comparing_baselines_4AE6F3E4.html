Руководство по инструменту: Сравнение контрольных версий с помощью Rational ClearCase
В этой памятке по инструменту описывается сравнение двух контрольных версий компонента UCM с обозревателем структуры компонентов Rational ClearCase Component Tree Browser.
Инструмент: Rational ClearCase
Взаимосвязи
Связанные элементы
  • Обновите рабочую область
  • Создание отчета о состоянии конфигурации
Основное описание

Обзор

В ClearCase контрольная версия UCM обычно представляет стабильную конфигурацию компонента.  Контрольная версия определяет действия и одну версию каждого элемента, видимого в одном или нескольких компонентах.  Обозреватель структуры компонентов ClearCase Component Tree Browser является графическим интерфейсом пользователя, отображающим хронологию контрольных версий компонента. С его помощью можно сравнивать содержимое двух контрольных версий.

Эта памятка по инструменту применима к Microsoft Windows.

Этапы работы с инструментами

Для сравнения двух контрольных версий в проекте UCM выполните следующие действия:

  1. Запустите ClearCase Explorer
  2. Откройте обозреватель структуры компонентов для выбранного компонента
  3. Выберите контрольную версию и параметры сравнения

Дополнительная информация приведена в руководстве ClearCase Управление программными проектами.

1. Запустите ClearCase Explorer

Для поиск компонента, хронологию контрольных версий для которого требуется просмотреть, выполните следующие действия:

  • На панели задач Windows выберите кнопку Пуск > Программы > Rational Software > Rational ClearCase > ClearCase Explorer.

2. Откройте обозреватель структуры компонентов для выбранного компненнта

  • Выберите Обзор контрольных версий во всплывающем меню компонента для запуска обозревателя структуры компонентов ClearCase.
  • Щелкните на папке Компоненты проекта для просмотра списка компонентов.
  • Щелкните правой кнопкой на компоненте чтобы открыть контекстное меню.
  • Выберите Обзор контрольных версий для запуска обозревателя структуры компонентов.

Обратитесь к разделу Обозреватель структуры компонентов ClearCase электронной справки ClearCase.

3. Выберите контрольную версию и параметры сравнения

  • Выберите одну из показываемых контрольных версий в качестве основы для сравнения.
  • В меню Инструменты выберите один из элементов меню Сравнить.... Появится приглашение задать вторую контрольную версию для сравнения.
  • Можно просмотреть различия действий или версий. Выберите нужную вкладку.
