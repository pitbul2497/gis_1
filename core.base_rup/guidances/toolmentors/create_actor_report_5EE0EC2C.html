Обзор 

SoDA автоматизирует создание отчета для повышения скорости и точности его создания. Создать отчет субъекта можно в версии Microsoft® Word® или Adobe® FrameMaker® SoDA.

Связанная информация Rational Unified Process:

  •  Отчет: субъект
  • Продукт работы: субъект

Эта памятка по инструменту применима для Windows 2000, NT 4.0, Windows XP, Solaris или HP-UX.

Для создания отчета субъекта с помощью Rational SoDA выполните процедуру, предназначенную для вашей версии продукта:

  • Применение Rational SoDA/Word для создания отчета субъекта
  • Применение Rational SoDA/FrameMaker для создания отчета субъекта

Применение Rational SoDA/Word для создания отчета субъекта

  1. Выберите на диаграмме прецедентов в Rational Rose субъект для документирования.
  2. Выберите опцию Отчет > Отчет SoDA.
  3. При появлении в SoDA списка доступных отчетов выберите Отчет субъекта Rational Unified Process. Если этот отчет отсутствует в списке, вернитесь назад и проверьте, что вы выбрали субъект на диаграмме прецедентов.
  4. Нажмите OK для создания отчета.

Применение Rational SoDA/FrameMaker для создания отчета субъекта

  1. На панели кнопок FrameMaker выберите Создать. Дважды щелкните на SoDA, затем дважды щелкните на RoseDomain и выберите шаблон RUPActorReport.fm.
  2. Измените коннектор, введите имя модели, пакета и субъекта.
  3. Выберите Файл > Сохранить как для сохранения шаблона в личной папке или в каталоге проекта.
  4. Выберите SoDA > Создать документ.
  5. Просмотрите созданный документ.
