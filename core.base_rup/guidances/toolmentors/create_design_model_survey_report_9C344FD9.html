Руководство по инструменту: Создание отчета исследования дизайн-модели с помощью Rational SoDA
В этой памятке по инструменту описывается применение Rational SoDA для создания отчета исследования дизайн-модели.
Инструмент: Rational SoDA
Основное описание

Обзор

SoDA автоматизирует создание отчета для повышения скорости и точности его создания. Создать отчет исследования дизайн-модели можно в версии Microsoft® Word® или Adobe® FrameMaker® SoDA. Для создания этого отчета SoDA собирает только элементы, заданные в дизайн-модели в Rational Rose. Это применимо только ели модель Rose соответствует соглашению об именах и структуре для дизайн-модели.

Связанная информация Rational Unified Process:

  •  Отчет: исследование дизайн-модели
  • Продукт работы: дизайн-модель  

Эта памятка по инструменту применима для Windows 2000, NT 4.0, Windows XP, Solaris или HP-UX.

Для создания отчета исследования дизайн-модели с помощью Rational SoDA выполните процедуру, предназначенную для вашей версии продукта:

  • Применение Rational SoDA/Word для создания отчета исследования дизайн-модели
  • Применение Rational SoDA/FrameMaker для создания отчета исследования дизайн-модели

Применение Rational SoDA/Word для создания отчета исследования дизайн-модели 

  1. Выберите в Rational Rose опцию Отчет > Отчет SoDA.
  2. При появлении в SoDA списка доступных отчетов выберите Отчет исследования дизайн-модели Rational Unified Process.
  3. Нажмите OK для создания отчета.

Применение Rational SoDA/FrameMaker для создания отчета исследования дизайн-модели

  1. На панели кнопок FrameMaker выберите Создать. Дважды щелкните на SoDA, затем дважды щелкните на RoseDomain и выберите RUPDesignModelSurvey.fm.
  2. Измените коннектор, указав имя файла модели.
  3. Выберите Файл > Сохранить как для сохранения шаблона в личной папке или в каталоге проекта.
  4. Выберите SoDA > Создать документ.
  5. Просмотрите созданный документ.

Когда в следующий раз потребуется создать тот же документ, просто откройте документ и выберите SoDA>Создать документ.

Соглашение о структуре и именах для модели Rose


Элементы из дизайн-модели извлекаются в отчет исследования дизайн-модели.
