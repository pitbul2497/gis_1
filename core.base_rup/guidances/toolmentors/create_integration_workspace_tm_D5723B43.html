Основное описание

Обзор

Рабочая область интеграции схожа с "частной рабочей областью" реализующего класса, в которой можно разрабатывать и из менять код содержащей области. Рабочая область интеграции является пространством, в котором интеграторы ПО проверяют возможность компоновки и совместной работы в качестве единого продукта разработанных по отдельности компонентов.

Эта памятка по инструменту применима к Microsoft Windows.

Этапы работы с инструментом

Для создания рабочей области интеграции и компоновки с помощью ClearCase выполните следующие действия:

  1. На панели задач Windows перейдите к мастеру создания панелей, выбрав Пуск > Программы > Rational Software > Rational ClearCase > Создать панель.
  2. С помощью мастера создания панелей создайте рабочую область (панель ClearCase) и настройте ее для выбора предложенных изменений разработки. Вас попросят ввести следующую информацию:

    • имя панели
    • расположение панели
    • прочие сведения, зависящие от конфигурации сайта (мастер создания панели проведет вас через все необходимые шаги)

 Обратитесь к электронной справке, доступной в мастере создания панелей для получения дополнительных сведений о создании панели.

 Общие сведения о панелях ClearCase, включая сведения о конфигурации, доступны в руководстве ClearCase Разработка ПО.
