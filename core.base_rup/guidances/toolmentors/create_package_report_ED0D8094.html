Руководство по инструменту: Создание отчета пакета с помощью Rational SoDA
В этой памятке по инструменту описывается применение Rational SoDA для создания отчета пакета.
Инструмент: Rational SoDA
Основное описание

Обзор

SoDA автоматизирует создание отчета для повышения скорости и точности его создания. Создать отчет пакета можно в версии Microsoft® Word® или Adobe® FrameMaker® SoDA.

Связанная информация Rational Unified Process:

  •  Отчет: пакет/подсистема эскиза
  • Продукт работы: пакет эскиза
  • Продукт работы: подсистема эскиза  

Эта памятка по инструменту применима для Windows 2000, NT 4.0, Windows XP, Solaris или HP-UX.

Для создания отчета пакета с помощью SoDA выполните процедуру, предназначенную для вашей версии продукта:

  • Применение Rational SoDA/Word для создания отчета пакета
  • Применение Rational SoDA/FrameMaker для создания отчета пакета

Применение Rational SoDA/Word для создания отчета пакета

  1. Выберите на диаграмме классов в Rational Rose пакет для документирования.
  2. Выберите опцию Отчет > Отчет SoDA.
  3. При появлении в SoDA списка доступных отчетов выберите Отчет пакета Rational Unified Process. Если этот отчет отсутствует в списке, вернитесь назад и проверьте, что вы выбрали пакет на диаграмме классов.
  4. Нажмите OK для создания отчета.

Применение Rational SoDA/FrameMaker для создания отчета пакета

  1. На панели кнопок FrameMaker выберите Создать. Дважды щелкните на SoDA, затем дважды щелкните на RoseDomain и выберите шаблон RUPPackageReport.fm.
  2. Измените коннектор, введите имя файла модели и имя пакета.
  3. Выберите Файл > Сохранить как для сохранения шаблона в личной папке или в каталоге проекта.
    Вы можете изменить имя шаблона, чтобы оно отражало имя пакета, например, AuthenticationPackage.fm.
  4. Выберите SoDA > Создать документ.
  5. Просмотрите созданный документ.

Когда в следующий раз потребуется создать тот же документ, просто откройте документ и выберите SoDA>Создать документ.
