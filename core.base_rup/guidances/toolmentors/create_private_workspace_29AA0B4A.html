Основное описание

Обзор

В этой памятке по инструменту описывается процесс создания частной рабочей области для личной работы. "Частная рабочая область" отделана от рабочих областей других участников команды. Она является местом, где разработчик вносит изменения в систему не видя ил не влияя на изменения, вносимые другими разработчиками.

В модели UCM ClearCase частные рабочие области, называемые рабочими областями разработки, позволяют работать с операциями UCM в изоляции от остальных частей  проекта. Обычно каждый участник проекта создает собственную рабочую область разработки при подключении к работе над проектом UCM. Частные рабочие области также можно создать в любое время при разработке проекта UCM.

Рабочие области разработки UCM состоят из потоков связанных панелей. Потоки UCM определяют какие версии элементов отображаются в панелях, настроенных этим потоком. Потоки также поддерживают список контрольных версий и операций UCM. Панели предоставляют рабочие области для таких задач, как изменение и компиляция исходных файлов, форматирование документов и т.д.

Эта памятка по инструменту применима к Microsoft Windows.

Этапы работы с инструментом

Описанные ниже действия касаются создания рабочей области разработки при подключении к проекту UCM. Общими шагами являются:

  1. Запуск мастера подключения к проекту
  2. Определение сведений для потока и панели
  3. Присоединение панели к потоку

  См. раздел Мастер подключения к проекту электронной справки ClearCase для обзора мастера. Подробная справка по каждому шагу мастера приведена в справке в каждом соответствующем окне.

Дополнительные сведения о панелях и потоках UCM приведены в следующих разделах электронной справки ClearCase.

  •  О потоках 
  •  О панелях

1. Запустите мастер подключения к проекту

  1. На панели задач Windows выберите кнопку Пуск > Программы > Rational Software > Rational ClearCase > Project Explorer.
  2. В ClearCase Project Explorer перейдите на вкладку Панель инструментов > UCM > Подключиться к проекту. Откроется мастер подключения к проекту.
  3. Выберите проект, к которому вы хотите подключиться, и нажмите кнопку Далее для перехода к следующему этапу выполнения мастера.

2. Задайте сведения о потоке и панели

Укажите имя потока, имя панели, место хранения панели и тип панели (моментальная копия) для рабочей области разработки. Мастер подключения к проекту ClearCase предоставит сведения по умолчанию для этих значений.

Примечание: мастер подключения к проекту также предложит ввести информацию, требуемую для создания панели для рабочей области интеграции проекта.

  Дополнительные сведения о хранении панелей приведены в разделе Выбор расположения каталога панели моментальной копии электронной справки ClearCase.

3. Присоединение панели к потоку

Мастер подключения к проекту попросит вас подтвердить введенные значения и присоединит указанные панель и поток разработки. Нажмите OK для подтверждения.

  Дополнительные сведения о взаимосвязи панелей и потоков приведены в электронной справке ClearCase в разделе > Присоединение новой панели к существующему потоку.
