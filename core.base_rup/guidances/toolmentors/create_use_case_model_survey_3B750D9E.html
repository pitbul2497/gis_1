Обзор

SoDA автоматизирует создание отчета для повышения скорости и точности его создания. Создать отчет исследования модели прецедентов можно в версии Microsoft® Word® или Adobe® FrameMaker® SoDA. Это применимо только если модель Rational Rose соответствует соглашению об именах и структуре для модели прецедентов.

Связанная информация Rational Unified Process:

  •  Отчет: исследование модели прецедентов
  • Продукт работы: модель прецедентов  

Эта памятка применима к Windows 2000, NT 4.0, Windows XP, Solaris или HP-UX.

Для создания отчета исследования модели прецедентов помощью SoDA выполните процедуру, предназначенную для вашей версии продукта:

  • Применение Rational SoDA/Word для создания отчета модели прецедентов
  • Применение Rational SoDA/FrameMaker для создания отчета исследования модели прецедентов

Применение Rational SoDA/Word для создания отчета модели прецедентов

  1. Выберите в Rational Rose опцию Отчет > Отчет SoDA.
  2. При появлении в SoDA списка доступных отчетов выберите Отчет модели прецедентов Rational Unified Process.
  3. Нажмите OK для создания отчета.

Применение Rational SoDA/FrameMaker для создания отчета исследования модели прецедентов

  1. На панели кнопок FrameMaker выберите Создать. Дважды щелкните на SoDA, затем дважды щелкните на RoseDomain и выберите шаблон RUPUseCaseModelSurvey.fm.
  2. Измените коннектор, указав имя файла модели.
  3. Выберите Файл > Сохранить как для сохранения шаблона в личной папке или в каталоге проекта.
  4. Выберите SoDA > Создать документ.
  5. Просмотрите созданный документ.

Когда в следующий раз потребуется создать тот же документ, просто откройте документ и выберите SoDA>Создать документ.
