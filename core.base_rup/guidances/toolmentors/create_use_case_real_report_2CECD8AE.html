Руководство по инструменту: Создание отчета реализации прецедента с помощью Rational SoDA
В этой памятке по инструменту описывается применение Rational SoDA для создания отчета реализации прецедента.
Инструмент: Rational SoDA
Основное описание

Обзор

SoDA автоматизирует создание отчета для повышения скорости и точности его создания. Создать отчет реализации прецедента можно в версии Microsoft® Word® или Adobe® FrameMaker® SoDA.

Связанная информация Rational Unified Process:

  •  Отчет: реализация прецедентов
  • Продукт работы: реализация прецедента

Эта памятка применима к Windows 2000, NT 4.0 или Windows XP, Solaris, либо HP-UX.

Для создания отчета реализации прецедента с помощью SoDA выполните процедуру, предназначенную для вашей версии продукта:

  • Применение Rational SoDA/Word для создания отчета реализации прецедента
  • Применение Rational SoDA/FrameMaker для создания отчета реализации прецедента

Применение Rational SoDA/Word для создания отчета реализации прецедента 

  1. Выберите на диаграмме вариантов в Rational Rose вариант для документирования.
  2. Выберите опцию Отчет > Отчет SoDA.
  3. При появлении в SoDA списка доступных отчетов выберите Отчет реализации прецедента Rational Unified Process.
    Если этот отчет отсутствует в списке, вернитесь назад и проверьте, что вы выбрали вариант на диаграмме.
  4. Нажмите OK для создания отчета.

Применение Rational SoDA/FrameMaker для создания отчета реализации прецедента

  1. На панели кнопок FrameMaker выберите Создать. Дважды щелкните на SoDA, затем дважды щелкните на RoseDomain и выберите отчет RUPUseCaseRealizationReport.fm.
  2. Измените коннектор, введите имя модели, пакета и варианта.
  3. Выберите Файл > Сохранить как для сохранения шаблона в личной папке или в каталоге проекта.
    Можно изменить имя шаблона, чтобы оно отражало имя реализации прецедента, например, MaintainLoansReport.fm.
  4. Выберите SoDA > Создать документ.
  5. Просмотрите созданный документ.

Когда в следующий раз потребуется создать тот же документ, просто откройте документ и выберите SoDA>Создать документ.
