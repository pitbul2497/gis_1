Основное описание

Обзор

SoDA автоматизирует создание отчета для повышения скорости и точности его создания. Создать отчет прецедента можно в версии Microsoft® Word® или Adobe® FrameMaker® SoDA.

Связанная информация Rational Unified Process:

  •  Отчет: спецификация прецедента
  • Продукт работы: процедент

Эта памятка применима к Windows 2000, NT 4.0, Windows XP, Solaris или HP-UX.

Для создания отчета прецедента с помощью SoDA выполните процедуру, предназначенную для вашей версии продукта:

  • Применение Rational SoDA/Word для создания отчета прецедента
  • Применение Rational SoDA/FrameMaker для создания отчета прецедента

Применение Rational SoDA/Word для создания отчета прецедента

  1. Выберите на диаграмме вариантов в Rational Rose вариант для документирования.
  2. Выберите опцию Отчет > Отчет SoDA.
  3. При появлении в SoDA списка доступных отчетов выберите Отчет прецедента Rational Unified Process.
    Если этот отчет отсутствует в списке, вернитесь назад и проверьте, что вы выбрали вариант на диаграмме вариантов.
  4. Нажмите OK.
  5. При появлении приглашения UseCaseSpecification укажите файл Word, добавляемый в этот отчет.
  6. Нажмите OK для создания отчета.

2. Применение Rational SoDA/FrameMaker для создания отчета прецедента

  1. На панели кнопок FrameMaker выберите Создать. Дважды щелкните на SoDA, затем дважды щелкните на RoseDomain и выберите шаблон RUPUseCaseReport.fm.
  2. Измените первый коннектор и введите имя модели, пакета и варианта.
  3. Выберите Файл > Сохранить как для сохранения шаблона в личной папке или в каталоге проекта. Может потребоваться изменить имя шаблона, чтобы оно отражало имя варианта, например, MaintainLoansReport.fm.
  4. Выберите SoDA > Создать документ.
  5. При появлении приглашения UseCaseSpecification укажите файл FrameMaker, добавляемый в этот отчет.
  6. Нажмите OK для создания отчета.
  7. Просмотрите созданный документ.

Когда в следующий раз потребуется создать тот же документ, просто откройте документ и выберите SoDA>Создать документ.
