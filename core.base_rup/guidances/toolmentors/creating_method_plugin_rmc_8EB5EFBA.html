Руководство по инструменту: Создание встраиваемого модуля метода с помощью Rational Method Composer
В этой памятке по инструменту описывается создание встраиваемого модуля метода с помощью Rational Method Composer.
Инструмент: Rational Method Composer
Взаимосвязи
Связанные элементы
  • Настройка процесса разработки для проекта
  • Подготовка рекомендаций по проекту
  • Подготовка шаблонов для проекта
Основное описание

A Встраиваемый модуль метода используется для хранения в пакете любых изменений (добавлений, удалений, расширений, и т.д.)  в содержимом и процессах существующего метода.  

Сведения о создании встраиваемых модулей методов приведены в памятке "Создание собственного встраиваемого модуля метода".  Все памятки Rational Method Composer доступны через меню Справка>Памятка.

После создания встраиваемого модуля необходимо заполнить его содержимым:

  • Информация о разработке содержимого метода см. Памятку по инструменту: разработка содержимого метода с помощью Rational Method Composer.  
  • Информация о разработке процессов приведена в Памятке по инструменту: разработка содержимого метода с помощью Rational Method Composer.
