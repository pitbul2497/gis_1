Основное описание

Чем может быть полезен ClearQuest MultiSite для вашей компании?

ClearQuest MultiSite® добавляет в ClearQuest новые мощные возможности. С помощью MultiSite разработчики из разных мест могут использовать одну схему и базу пользователей. В каждом расположении (сайте) имеется копия (реплика) базы данных и ее соответствующего хранилища схем. В любое время сайт может заполнить внесенными в его копию изменениями прочие сайты, отправив пакеты обновления. Процесс обновления может быть автоматическим или запускаться вручную по команде.  

Конфликтующие изменения записей отслеживаются и устраняются схемой участников ClearQuest MultiSite. Большинство объектов ClearQuest имеют только одну копию, изменять запись могут только пользователи этой копии. MultiSite позволяет использовать общее управление записями, чтобы над одной записью могли работать несколько пользователей по очереди.  

С помощью инструментов синхронизации, поставляемых с ClearQuest MultiSite можно отправлять изменения между копиями так часто, как это позволяет соединение между сайтами. При наличии IP-соединений можно использовать механизмы отправки MultiSite для синхронизации копий автоматически с заданной высокой частотой, предоставляя пользователям других сайтов доступ ко все последним изменениям, внесенным на вашем сайте. При отсутствии IP-соединений можно выбрать другой способ синхронизации (например, электронную почту или ftp), подходящий для вашей реализации.

Нужен ли вам ClearQuest MultiSite?

ClearQuest также позволяет создавать отчеты с помощью существующих запросов. Отчет ClearQuest состоит из запроса, вставленного в формат отчета, созданный инструментом другой фирмы. Формат отчета создается ролью, такой как руководитель, затем формат отчета сохраняется в ClearQuest Workspace. Для создания отчета выберите формат отчета, затем укажите результаты каких запросов будут вставлены в формат отчета. Отчеты хранятся в рабочей области и доступны для просмотра в любое время.
