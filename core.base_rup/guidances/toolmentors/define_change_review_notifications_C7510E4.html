Руководство по инструменту: Определение уведомлений об изменении и проверке с помощью Rational ClearQuest
В этой памятке по инструменту описывается настройка Rational ClearQuest& для уведомления пользователей или групп пользователей об изменении состояния запроса на изменение.

Основное описание

Обзор

ClearQuest использует электронную почту двумя способами: для отправки изменений в базу данных (от пользователя к базе данных) и для рассылки уведомлений (от базы данных к пользователю).

Отправка изменений по электронной почте: администратор ClearQuest использует программу просмотра электронной почты Rational E-Mail Reader для предоставления пользователям возможности отправлять и изменять записи через электронную почту. Для обеспечения доступности этой функции необходимо обеспечить постоянную работу почтового сервера Rational в той же системе, в которой выполняются базы данных ClearQuest.

Рассылка уведомлений по электронной почте: администратор ClearQuest может настроить ClearQuest для отправки по электронной почте уведомлений пользователям об изменении запросов на изменение. Администратор использует тип записи E-Mail Rules для определения условий уведомлений по электронной почте, получателей и содержимого. Кроме того, администратор или пользователи могут настроить свои клиентские системы для получения уведомлений по электронной почте. Даже Web-клиенты ClearQuest могут получать уведомления по электронной почте.

Этапы работы с инструментами

1. Настройте отправку изменений с помощью ClearQuest Designer

Для просмотра электронной справки по ClearQuest Designer выберите в меню пункт Справка > Содержимое и индекс > Администрирование электронной почты ClearQuest > Включить отправку изменений по электронной почте.

2. Настройка рассылки уведомлений по электронной почте с помощью ClearQuest Designer

Для просмотра справки ClearQuest Designer выберите в меню пункт Справка > Содержимое и индекс > Администрирование электронной почты ClearQuest > Включение автоматической рассылки уведомлений по электронной почте.

3. Получение уведомлений по электронной почте клиентом ClearQuest

Для просмотра справки по ClearQuest выберите пункт меню Справка > Содержимое и индекс > Администрирование электронной почты ClearQuest > Включение функции электронной почты клиента ClearQuest.

4. Настройте рассылку уведомлений пл электронной почте для Web-клиентов ClearQuest
