Обзор 

Эта памятка по инструменту поддерживает выполнение описания распределения с помощью Rational XDE Developer - Java Platform Edition.

  • Шаги, описанные в этой памятке по инструменту, соответствуют этапам выполнения определенных задач.
  • Ссылки на разделы электронной справки XDE помечены с помощью значка .
  • В этой памятке по инструменту выполняются следующие шаги:
    • Анализ требований распределения
    • Определение сетевой конфигурации
    • Выделение системных элементов узлам  

Анализ требований распределения

Для данного шага определенные руководства Rational XDE (TM) отсутствуют.

Определение сетевой конфигурации

Уточните диаграмму развертывания, созданную в Задаче: анализ архитектуры (Памятка по инструменту: выполнение архитектурного анализа с помощью Rational XDE).

Справочник: Работа с диаграммами развертывания

Выделение системных элементов узлам

Добавьте аннотации к диаграмме развертывания или добавьте дополнительные диаграммы для отображения логического развертывания (классов, подсистем или нитей управления, привязанных к узлам) и/или физического развертывания (файлов или наборов файлов, привязанных к узлам). Основные этапы перечислены ниже:

  1. Добавьте при необходимости дополнительные диаграммы развертывания. Справочник: Добавление диаграмм
  2. Добавьте в диаграмму узлы или экземпляры узлов. Справочник: Добавление форм к диаграммам
  3. Переместите элементы модели разработки или реализации в диаграмму. Справочник: Добавление форм к диаграммам Если развертываемый элемент еще не смоделирован, добавьте его при необходимости к соответствующей модели. (См. Рекомендации по структуре модели Rational XDE для получения информации о месте моделирования каждого типа элемента).
  4. Добавьте взаимосвязи развертывания между узлами и развертываемыми элементами. Справочник: Добавление взаимосвязей развертывания
