Идентификация процессов и нитей

При наличии параллелизма приложений следует идентифицировать нити управления, отображаемые в виде активных классов.

  1. Идентифицируйте пакеты, к которым принадлежат активные классы. Перейдите к этому пакету.
  2. Добавьте активный класс к диаграмме классов. См. Добавление форм к диаграммам .
  3. Пометьте класс как активный. См. Активация классов или объектов .
  4. Перейдите к реализации прецедентов , для которой нужен данный активный класс.
  5. Отобразите сложную связь между процессами и нитями с помощью диаграммы последовательностей. Рекомендации по созданию диаграмм последовательностей в виде части реализации прецедентов приведены в Памятке по инструменту: разработка прецедентов с помощью Rational XDE.
  6. Укажите тип параллелизма для операций. Определение типа параллелизма для операций Сведения об управляемых сообщениями EJB приведены в Памятке по инструменту: определение элементов разработки с помощью Rational XDE.

Идентификация жизненных циклов процесса

Добавьте диаграмму последовательностей для представления жизненных циклов процессов и нитей. Каждый процесс или нить должен отображаться на диаграммах последовательностей, создающих и уничтожающих их. Сведения о создании диаграмм последовательностей приведены в Памятке по инструменту: разработка прецедентов с помощью Rational XDE.

Привязка элементов разработки к нитям управления

С помощью взаимосвязей агрегирования можно показать состав активных классов и отобразить неактивные классы, выполняемые под управлением активных классов.

  1. Перейдите к пакету модель разработки, содержащему панель процессов. См. Рекомендации по структуре модели Rational XDE.
  2. Добавьте диаграмму классов к этому пакету. См. Добавление диаграмм .
  3. Переместите активные классы вместе с значимыми классами разработки, находящимися под управлением этих активных классов, на диаграмму классов. См. Добавление форм к диаграммам .
  4. Добавьте взаимосвязи ассоциаций. См. Добавление взаимосвязей ассоциаций .
  5. Задайте агрегирование. Определение типов взаимосвязей ассоциаций Справочник: Работа с диаграммами классов
