Руководство по инструменту: Разработка содержимого метода с помощью Rational Method Composer
В этой памятке описывается разработка элементов содержимого метода, таких как роли, задачи, продукты работы и рекомендации. Эта памятка по инструменту описывает создание нового содержимого метода, заполнения содержимого существующего метода и замены содержимого существующего метода.

Основное описание

Разработка содержимого метода заключается в создании и описании элементов содержимого метода (Роль, Задача, Продукт работы и  Рекомендация), а также их организации в пакете содержимогоs.  Также этот процесс подразумевает создание  Категорий и панелей.  

Все содержимое метода должно находиться во встраиваемом модуле метода.  Сведения о создании встраиваемого модуля метода приведены в Памятке по инструменту: создание встраиваемого модуля с помощью Rational Method Composer.

Для определения места создания элемента или для поиска элемента для заполнения или замены может потребоваться перемещаться по существующему содержимому метода.  Дополнительные сведения о навигации по содержимому метода приведены в Памятке по инструменту: навигация по содержимому метода с помощью Rational Method Composer.

Сведения о разработке элементов метода приведены в следующих памятках: "Добавление содержимого нового метода", "Заполнение содержимого существующего метода", "Замена содержимого существующего метода" и "Добавление указателя на метод или гиперссылки". Все памятки Rational Method Composer доступны через меню Справка>Памятка.
