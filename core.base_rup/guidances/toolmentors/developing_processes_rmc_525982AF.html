Основное описание

В Rational Method Composer (RMC) есть два типа процессов -- Шаблон возможностей  и Процесс доставки.  

Все процессы должны находиться во встраиваемом модуле метода.  Сведения о создании встраиваемого модуля метода приведены в Памятке по инструменту: создание встраиваемого модуля с помощью Rational Method Composer.

При создании процессов рекомендуется начать с одного из предоставляемых с RMC.  Особенно интересны следующие:

  • Предустановленные процессы для проектов различного размера (в пакете "Процессы доставки").
  • Предустановленные шаблоны возможностей, созданные специально для использования при  создании процессов доставки (в подпакете "Шаблоны процессов доставки" пакета "Процессы доставки"). Есть шаблон возможностей "Этапы RUP", который можно применить вначале для определения структуры процессов с этапами и затем имеются отдельные шаблоны возможностей для итерации для каждого этапа RUP, которые можно применить в этой структуре.

Сведения о применении процессов в контексте приведены в разделе Сведения о применении.

Дополнительные сведения о навигации по содержимому метода приведены в Памятке по инструменту: навигация по содержимому метода с помощью Rational Method Composer.

Сведения о создании процессов с помощью RMC приведены в памятке "Работа с процессами".  Все памятки RMC доступны через меню Справка>Памятка.
