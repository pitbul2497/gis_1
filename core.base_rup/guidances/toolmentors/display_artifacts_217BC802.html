Руководство по инструменту: Просмотр артефактов, относящихся к определенным объектам на диаграмме, с помощью Rational ProjectConsole
В этой памятке по инструменту описывается применения Rational ProjectConsole для навигации по определенным артефактам диаграммы, отображаемой Rational ProjectConsole.

Основное описание

Обзор

ProjectConsole может отображать как текст, так и графику. В случаях отображения диаграммы продукта работы, с помощью диаграммы можно немедленно перейти к Web-странице, созданной для элемента, показываемого на диаграмме. Например, на диаграмме модели может быть показано число пакетов или вариантов в виде графиков. В этой памятке по инструменту демонстрируется, что при щелчке на элемент, скажем пакет или вариант, отображаемый на графике, будет показана страница продукта работы.

ProjectConsole использует технологию аплетов Java для предоставления вам знакомых множественных форм навигации. При щелчке на тексте в узле управления структурой отображается страница продукта работы, связанной с этим узлом. Если развернуть узел управления структурой, щелкнув на значке плюса, будут показаны гиперссылки, отображаемые при щелчке на тексте узла, в виде дочерних узлов, которые можно использовать для дальнейшей навигации.

Эта памятка по инструменту применима при работе с Microsoft Internet Explorer (начиная с версии 5.5) или Netscape (начиная с версии 7.0).

Этапы работы с инструментами

Для отображения продуктов работы, связанных с определенными объектами, на диаграмме с помощью ProjectConsole выполните следующие действия:

  1. Запустите браузер.
  2. Введите URL установочного сайта ProjectConsole. Получить URL можно у администратора ProjectConsole. Появится окно входа в систему ProjectConsole.
  3. Введите допустимое ИД пользователя и пароль. Откроется обозреватель продуктов работы ProjectConsole.
  4. Разверните следующие узлы: ProjectConsole, Проекты ClassicsCD.com, Точка продаж (POS), Функциональные команды, Разработка, Версия 1.0, Итерация 1, Эскиз- Текущая панель
  5. Щелкните на гиперссылке Панель прецедентов, расположенной в правом фрейме ProjectConsole. Будет показан шаблон продукта работы прецедента со сведениями панели прецедента.
  6. Щелкните на графике в панели прецедента для просмотра страницы продукта работы. Например, щелкните на пакете Точка продаж для отображения содержимого пакета Точка продаж.
  7. При щелчке на любом элементе на странице продукта работы вы перейдете к дополнительной информации модели. Например, при щелчке на варианте Проверка пользователя будет показана страница продукта работы со сведениями о варианте проверки пользователя без пакета Точка продаж.

Примечание: Отчеты, показываемые ProjectConsole являются простыми шаблонами, созданными компоновщиком шаблонов Rational ProjectConsoleTemplateBuilder, в сводной панели ProjectConsole создаются опубликованные диаграммы.
