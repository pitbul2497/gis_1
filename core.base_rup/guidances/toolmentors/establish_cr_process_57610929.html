Основное описание

Обзор

ClearQuest является системой управления запросами изменений (CRM) для динамического и интерактивного процесса разработки ПО. С помощью ClearQuest можно управлять всеми типами действий изменения, связанными с разработкой ПО, включая запросы расширения, отчеты о дефектах и изменение документации.

Применение ClearQuest не ограничивается отслеживанием неполадок. ClearQuest Designer является инструментом администрирования для настройки схем ClearQuest для определенных потребностей вашей фирмы. Администратор задает тип информации, отправляемой, получаемой и просматриваемой в отчетах и диаграммах пользователем. Администратор также определяет какие пользователи какие действия могут выполнять над записями.

От использования ClearQuest выигрывают все члены вашей команды разработчиков:

  • Инженеры разработки смогут уделить больше времени написанию кода и меньше времени идентификации выполнимым сообщениям.
  • Инженеры тестирования смогут легче определять источник, состояние и решение всех запросов изменений.
  • Руководители и администраторы проекта смогут быстро узнавать о состоянии проектов с помощью подробных настраиваемых отчетов.
  • Удаленные члены команды смогут принимать участие в коллективной работе с помощью ClearQuest Web.

Этапы работы с инструментами

Настройка ClearQuest для создания процесса управления запросом изменения с помощью ClearQuest Designer состоит из четырех этапов.

  1. Создайте и измените схемы
  2. Создайте и измените базы данных
  3. Добавление пользователей и групп пользователей
  4. Создайте и измените код перехватчика

1. Создайте и измените схемы

Добавляет формы для отправки данных пользователями и изменения записей

См. электронную справку ClearQuest Designer Справка > Содержание и указатель:

  • Работа со схемами ClearQuest
  • Настройка схемы
  • Работа с формами

2. Создайте и измените базы данных

См. электронную справку ClearQuest Designer Справка > Содержание и указатель:

  • Управление базами данных
  • Работа со схемами ClearQuest

3. Добавление пользователей и групп пользователей, настройка и изменение разрешений для пользователей и групп

Добавляет параметры и изменяет разрешения для пользователей и групп

См. электронную справку ClearQuest Designer Справка > Содержание и указатель:

  • Применение защиты в Rational ClearQuest
  • Контролируемый пользователь

4. Создайте и измените код перехватчика

Добавляет дополнительные функции в ClearQuest разрешает взаимодействиям с внешними приложениями вызывать действия за пределами ClearQuest.

См. электронную справку ClearQuest Designer Справка > Содержание и указатель:

  • Применение перехватчиков для настройки потока операций
  • Примеры перехватчиков
