Руководство по инструменту: Оценка результатов выполнения тестового комплекта с помощью Rational TestFactory
В этой памятке описывается применение Rational TestFactory для измерения и оценки результатов выполнения тестового комплекта.
Инструмент: Rational TestFactory
Взаимосвязи
Связанные элементы
  • Анализ ошибки при тестировании
  • Выполнение комплекта тестов
  • Определение результатов тестирования
Основное описание

Обзор

После выполнения тестового комплекта в Rational TestFactory можно оценить результаты. Сведения о применении тестового комплекта для выполнения комплекта тестовых сценариев в TestFactory, приведены в Памятке по инструменту: выполнение комплекта тестовых сценариев. .

Эта памятка по инструменту применима для систем с Windows 98/2000/NT 4.0.

Для применение Rational TestFactory для оценки результатов выполнения комплекта тестовых сценариев выполните следующие действия:

  1. Просмотрите состояние выполнения тестовых сценариев комплекта
  2. Проанализируйте охват теста на базе кода

1. Просмотрите состояние выполнения тестовых сценариев в тестовом комплекте

Выбрав тестовый комплекте в левой панели главного окна Rational TestFactory, вы можете просмотреть на вкладке Состояние в правой панели результаты его выполнения. Для всех тестовых сценариев, перечисленных в тестовом комплекте, значение состояния указывает, успешно ли выполнен тестовый сценарий в TestFactory.

Если выполнить тестовый сценарий успешно не удалось, можно проанализировать протокол его выполнения в Rational TestManager.

   Обратитесь к следующим разделами электронной справки Rational TestFactory:

  • Вкладка Тестовый комплект
  • Просмотрите состояние выполнения тестового комплекта
  • Просмотрите протокол выполнения сценария

2. Проанализируйте охват теста на базе кода

При встраивании кода тестируемого приложения (AUT) перед запуском тестового комплекта можно просмотреть данные охвата теста на базе кода для тестового комплекта. Когда тестовый комплект является выбранным объектов в левой панели главного окна Rational TestFactory, на вкладке Охват в правой панели отображаются комбинированные значения охвата на базе кода для всех тестовых сценариев, выполняемых в тестовом комплекте.

Rational TestFactory вычисляет значения охвата теста на базе кода в качестве процентного значения исходного кода, затрагиваемого тестовыми сценариями, относительно всего исходного кода AUT. При наличии доступа к исходным файлам AUT можно воспользоваться браузером охвата в TestFactory для просмотра кода в исходном файле и изучения затрагиваемых тестами строк кода.

Можно также просмотреть значения охвата кода для отдельных тестовых сценариев, содержащихся в тестовом комплекте. Дважды щелкните на тестовом сценарии на вкладке Состояние тестового комплекта чтобы выбрать тестовый сценарий в левой панели. Дополнительные сведения приведены в Памятке по инструменту: оценка охвата теста.

   Обратитесь к следующим разделами электронной справки Rational TestFactory:

  • Встраивание тестируемого приложения
  • Вкладка Тестовый комплект
  • Просмотрите результаты охвата кода для тестового сценария
