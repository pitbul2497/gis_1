Руководство по инструменту: Вычисление тестового покрытия с помощью Rational TestFactory
В этой памятке по инструменту описывается применение Rational TestFactory для вычисления тестового покрытия на базе UI и на базе кода для сценария, автоматически созданного в TestFactory.
Инструмент: Rational TestFactory
Взаимосвязи
Связанные элементы
  • Анализ ошибки при тестировании
  • Анализ поведения среды выполнения
  • Выполнение комплекта тестов
  • Определение результатов тестирования
Основное описание

Обзор

Создав автоматически тестовые сценарии в Rational TestFactory можно просмотреть сведения о тестовом охвате для каждого сценария. Информация о применении Pilot для создания тестовых сценариев приведена в Памятке по инструменту: создание тестовых сценариев автоматически.

Эта памятка по инструменту применима для систем с Windows 98/2000/NT 4.0.

Для применения Rational TestFactory для вычисления тестового охвата выполните следующие действия:

  • Просмотрите сведения об охвате теста.

Просмотрите сведения о тестовом охвате

Когда в левой панели главного окна Rational TestFactory выбран тестовый сценарий, на вкладке Охват в правой панели содержится информация о тестовом охвате сценария.

Rational TestFactory вычисляет тестовый охват на базе UI в виде процентного значения от уникального управления приложением (или аплетом), затрагиваемого тестом, относительно числа управляющих элементов, доступных для Pilot.

Rational TestFactory вычисляет значение тестового охвата на базе кода в виде процентного значения относительно исходного кода, затрагиваемого тестовым сценарием, относительно всего исходного кода тестируемого приложения. При наличии доступа к исходным файлам AUT можно воспользоваться браузером охвата в TestFactory для просмотра кода в исходном файле и изучения затрагиваемых тестом строк кода.
