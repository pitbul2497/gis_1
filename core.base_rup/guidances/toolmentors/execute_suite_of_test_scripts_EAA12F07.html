Руководство по инструменту: Выполнение набора тестирования с помощью Rational TestFactory
В этой памятке по инструменту описывается применение TestFactory для создания и выполнения тестового комплекта.
Инструмент: Rational TestFactory
Взаимосвязи
Связанные элементы
  • Выполнение комплекта тестов
  • Реализация комплекта тестов
Основное описание

Обзор

"Тестовый комплект" является элементом Rational TestFactory, позволяющим размещать и выполнять комплект тестовых сценариев. Этот элемент TestFactory позволяет организовать тестовые сценарии, выполнить их в пакете в виде комплекта и проанализировать их тестовый охват на базе кода.

Эта памятка по инструменту применима для систем с Windows 98/2000/NT 4.0.

Для применения Rational TestFactory для выполнения комплекта тестовых сценариев выполните следующие действия:

  1. Создайте тестовый комплект тестовых сценариев
  2. Выполните тестовый комплект

1.   Создайте тестовый комплект тестовых сценариев

Объект тестового комплекта является контейнером, в который вы добавляете тестовые сценарии, которые требуется выполнит ь в виде пакета. Тестовый комплект может содержать тестовые сценарии записанные в Rational Robot, а также тестовые сценарии, созданные в Rational TestFactory.

  • Для создания тестового сценария в Robot обратитесь к памятке по инструменту: реализация тестовых сценариев с помощью Rational Robot.
  • Для создания тестовых сценариев в TestFactory обратитесь к Памятке по инструменту: реализация созданных тестовых сценариев с помощью Rational TestFactory.

Тестовые комплекты используются для организации и выполнения групп лучших сценариев, дефектных сценариев и сценариев, созданных для определенной компоновки тестируемого приложения (AUT). Добавляемые в комплект тестовые сценарии остается привязан к объекту тестовый сценарий. Тестовый комплект можно выполнить в качестве тестового комплекта регрессии для будущих компоновок AUT.

Rational TestFactory предоставляет два способа создания тестового комплекта:

  • Можно разрешить TestFactory создавать тестовый комплект автоматически с выбранными тестовыми сценариями.
  • Можно вручную добавлять объекта Тестовый сценарий в схему приложений и затем заполнять тестовый комплект тестовыми сценариями.

При необходимости проанализировать тестовый охват на базе кода для тестовых сценариев комплекта следует встроить код в AUT перед выполнением тестового комплекта. Rational TestFactory вычисляет тестовый охват на базе кода для тестовых сценариев только если приложение встроено. €

2.   Выполнение тестового комплекта

При выполнении тестового комплекта Rational TestFactory выполняет все тестовые сценарии комплекта по порядку их расположения в списке тестовых комплектов. Если порядок расположения тестовых сценариев важен, можно изменить его перед выполнением тестового комплекта.
