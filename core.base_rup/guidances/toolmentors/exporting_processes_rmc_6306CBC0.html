Основное описание

Есть два типа процессов в Rational Method Composer (RMC) -- Шаблон возможностейs  и Процесс доставки.  

Пред публикацией процесс следует создать.  Дополнительная информация о создании процесса приведена в Памятке по инструменту: разработка процессов с помощью Rational Method Composer.

После создания процессы можно экспортировать из RMC в инструмент планирования, где их можно использовать для управления проектом.

Для экспорта существующего процесса выберите "Файл>Экспорт", выберите "Шаблон проекта"  и следуйте приглашениям.  Дополнительная информация об экспорте процесса с помощью  RMC приведена в разделе электронной справки "Экспорт процессов и содержимого методов в инструмент планирования".  Вся электронная справка по RMC доступна через пункт меню Справка.
