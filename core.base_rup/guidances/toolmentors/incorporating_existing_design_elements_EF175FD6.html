Обзор 

Эта памятка по инструменту поддерживает элементы объединения разработки с Rational XDE Developer - Java Platform Edition.

  • Шаги, описанные в этой памятке по инструменту, соответствуют этапам выполнения определенных задач.
  • Ссылки на разделы электронной справки XDE помечены с помощью значка .
  • В памятке описаны следующие действия:
    • Определите возможности повторного применения
    • Выполните обратный инжиниринг для компонентов и баз данных
    • Обновите организацию модели разработки
    • Обновите логическую панель

Определите возможности повторного применения

Найдите существующие подсистемы или компоненты, предлагающие схожие интерфейсы. Некоторые ресурсы могли быть упакованы для намеренного повторного применения с помощью Reusable Asset Specification (RAS). При создании в Rational XDE такие ресурсы можно просматривать и загружать в XDE. См. Приступаем к работе с ресурсами RAS .

Выполните обратный инжиниринг для компонентов и баз данных

Для ресурсов, в которых отсутствует модель XDE, может оказаться полезным выполнение обратного инжиниринга для модели для лучшего понимания процесса разработки. Если ресурс окажется доступным для повторного применения, модель XDE становится частью общей модели разработки. См. Обратный инжиниринг .

Для ресурсов, управляемых вашей компанией, можно будет внести незначительные изменения в интерфейсы кандидатов, улучшающие соответствие с нужным интерфейсом. Для ресурсов, неуправляемых вашей компанией, можно создать адаптер или прикрепить через мост элементы разработки, привязывающие требуемый интерфейс к интерфейсам, предоставляемым повторно используемыми ресурсами. См. Памятка по инструменту: определение элементов разработки для получения рекомендаций по созданию интерфейсов и классов.

Обновите организацию модели разработки

Упорядочьте при необходимости элементы разработки. См. Группировка элементов модели в пакетах . Также см. информационный бюллетень Рекомендации по структуре модели Rational XDE.

Обновите логическую панель

При наличии отдельной логической панели необходимо обслуживать ее. См. информационный бюллетень Рекомендации по структуре модели Rational XDE.
