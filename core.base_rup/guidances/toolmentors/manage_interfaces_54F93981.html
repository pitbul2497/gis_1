Обзор

Для управления интерфейсами системы используется один или несколько пакетов. Для создания этих пакетов см. Памятку по инструменту: управление моделью разработки.

Для применения Rational Rose для создания интерфейсов выполните следующие действия:

  1. Создайте интерфейс
  2. Создайте операции интерфейса

1. Создайте интерфейс

Интерфейс задает видимые снаружи операции класса и/или компонента и не имеет собственной реализации. Обычно интерфейс указывает только определенную часть поведения класса или компонента. Интерфейсы принадлежат логическому представлению, но могут присутствовать и в диаграммах класса, варианта и компонента.

Дополнительная информация приведена в разделе Интерфейс электронной справки Rational Rose.

2. Создайте операции интерфейса

Для создания интерфейса для интерфейса изучите раздел Памятка по инструменту: управление классами, используйте те же действия, что и для класса, но примените их к интерфейсу.
