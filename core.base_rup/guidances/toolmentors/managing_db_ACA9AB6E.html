Сравнение и синхронизация

Rational XDE сравнивает модель данных с существующим файлом сценария DDL или схемой базы данных. (См. Сравнение модели данных .) Затем можно синхронизировать различия между моделью и DDL или базой данных. (См. Синхронизация модели данных .) Запустить этот процесс сравнения и синхронизации можно с помощью мастера синхронизации модели данных и базы данных. (См. Сравнение и синхронизация модели данных .)

Rational XDE требует выполнения сравнения и синхронизации модели данных на уровне базы данных. Схема сравниваемой DDL или базы данных должна иметь то же имя, что и база данных модели данных. Также необходимо сравнивать с DDL или базой данных, использующей ту же систему управления базами данных (DBMS), что и база данных модели данных.

Обратитесь к разделу электронной справки Rational XDE Сравнение и синхронизация модели данных .

Для выполнения сравнения и синхронизации необходимо соединение с базой данных. В следующих разделах электронной справки Rational XDE содержатся сведения об установке соединения с базой данных для определенной DBMS:

  • Соединения с базой данных Oracle
  • Соединения с базой данных SQL Server
  • Соединения с базой данных Sybase
  • Соединения с базой данных DB2
