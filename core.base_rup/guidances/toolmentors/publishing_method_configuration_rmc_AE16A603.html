Основное описание

Конфигурация метода является пользовательским подмножеством Библиотеки методов , хранящимся в Rational Method Composer (RMC).

Пред публикацией конфигурацию следует создать.  Сведения о создании конфигурации метода приведены в Памятке по инструменту: создание конфигурации метода с помощью Rational Method Composer.

Для публикации существующей конфигурации выберите "Конфигурация>Опубликовать" и следуйте появившимся инструкциям.  Д ля получения дополнительных сведений о публикации конфигурации метода с помощью  RMC обратитесь к разделу "Публикация конфигурации метода в Web" электронной справки.  Вся электронная справка по RMC доступна через пункт меню Справка.
