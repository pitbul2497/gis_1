Руководство по инструменту: Публикация Web-версий моделей Rational Rose с помощью Web Publisher
В этой памятке по инструменту описывается применение Rational Rose Web Publisher для создания web-версий (HTML) модели Rose, доступной остальным для просмотра через браузер, такой как Microsoft Internet Explorer или Netscape Navigator.

Основное описание

Обзор

Публикуя web-версии моделей Rose вы разрешаете остальным разработчикам асинхронно просматривать, выполнять поиск и перемещаться по моделям. Например, можно опубликовать последовательные итерации разрабатываемой модели для просмотра или обмена информацией. Rose Web Publisher воссоздает элементы модели Rose, включая диаграммы, классы, пакеты, взаимоотношения, атрибуты и операции. После публикации гипертекстовые ссылки позволяют просматривать модели, как это делается в Rose. Вы можете управлять содержимым публикации в Rose Web Publisher с помощью различных опций. Например, можно выбрать публикуемые представления моделей, объем добавляемых данных, используемую нотацию и графический формат диаграмм Rose. Обратите внимание, что если варианты модели содержат прикрепленную документацию варианта RequisitePro, Rose Web Publisher сохраняет документы и отображает их HTML-версии на опубликованных страницах.

Этапы работы с инструментами

Есть два способа опубликовать модель:

  1. С помощью диалоговых окон Web Publisher
  2. С помощью интерфейса командной строки

1.  С помощью окон Web Publisher

Простейшим способом создания Web-моделей является применение встраиваемого модуля Web Publisher, доступного в меню Инструменты Rational Rose. В окне Web Publisher имеется браузер, позволяющий выбрать элементы модели для публикации. Помимо этого можно указать, должны ли публикуемые классы содержать сведения о суперклассе, и какой графический формат следует использовать для диаграмм (.jpg или .gif). Функция предварительного просмотра позволяет запустить браузер по умолчанию и просмотреть опубликованную модель непосредственно из Rose Web Publisher. 

Для публикации модели выполните следующие действия:

  1. В Rational Rose откройте публикуемую модель.
  2. В меню Инструменты выберите Web Publisher.
  3. С помощью браузера выберите публикуемые элементы модели, задайте опции публикации.
  4. Нажмите кнопку Опубликовать когда будете готовы к публикации модели.  
  5. Нажмите кнопку Предварительный просмотр чтобы открыть web-браузер по умолчанию и просмотреть опубликованную модель.  

Дополнительная информация о Web Publisher приведена в электронной справке.

2.  Использование интерфейса командной строки

Rational Rose Web Publisher предоставляет интерфейс командной строки, позволяющий опубликовать модели с помощью обработчика пакетных файлов. Особенно удобна эта функция при необходимости автоматической публикации моделей по расписанию. Указать объем публикуемых сведений можно задав параметры публикации в файле инициализации. При вызове обработчика Web Publisher вы задаете имя файла инициализации. Обработчик rosewpbatch.exeрасположен в подкаталоге rosewp установочного каталога Rose. Ниже приведен синтаксис вызова обработчика пакетных файлов: 

rosewpbatch.exe имя_файла_инициализации.ini 

Пример: c:\rose\rosewp\rosewpbatch.exe c:\mybatchfiles\testbatch.ini

Где имя_файла_инициализации.ini является именем созданного вами файла инициализации. Добавьте полный путь к файлу .ini в вызов обработчика. Если обработчик обнаружит ошибки, они будут записаны в файл rosewpbatch.err , расположенный в том же каталоге, что и сам обработчик. Для просмотра содержимого используйте любой текстовый редактор.

Обратитесь к электронной справке Web Publisher для получения сведений о параметрах и синтаксисе файла инициализации.
