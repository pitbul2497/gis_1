Руководство по инструменту: Создание отчета о тенденциях неполадок и их состоянии с помощью Rational ClearQuest
В этой памятке по инструменту описывается создание диаграммы Rational ClearQuest(R) для просмотра тенденций неполадок и и создания отчета о состоянии неполадок.
Инструмент: Rational ClearQuest
Взаимосвязи
Связанные элементы
  • Определение результатов тестирования
  • Оценка и обоснование оценки качества
  • Создание отчета о состоянии конфигурации
Основное описание

Обзор

ClearQuest предоставляет предустановленные диаграммы и отчеты, позволяющие ролям, таким как менеджер, просматривать состояние проекта. Также облегчается изменение диаграмм и отчетов и создание новых.

Диаграммы 

Диаграммы Rational ClearQuest предоставляют графическое отображение данных запросов изменений. ClearQuest поддерживает три типа диаграмм: распределения, тенденций и давности.

Диаграммы распределения показывают, сколько записей помещается в категориях или критериях запроса. Например, менеджер проекта может создавать диаграммы для отображения:

  • Текущего состояния группы записей
  • Кому присвоено больше/меньше всех запросов изменения
  • Какие записи обладают максимальным приоритетом

Диаграммы тенденций показывают:

  • Сколько записей перемещено в выбранное состояние по дням, неделям или месяцам.
  • Скорость отправки, обработки или перемещения записей в другие состояния.

Диаграммы давности показывают сколько записей находятся в выбранном состоянии и как долго. Диаграммы давности отображают такие сведения как:

  • Сколько неполадок открыты менее недели.
  • Сколько неполадок открыты более трех недель.
  • Сколько неполадок отложены более двух месяцев.

На диаграммах можно отобразить давность и тенденции для всех записей на основе состояния (неполадок, запросов расширения, запросов описания и т.д.).

Отчеты

ClearQuest также позволяет создавать отчеты с помощью существующих запросов. Отчет ClearQuest состоит из запроса, вставленного в формат отчета, созданный инструментом другой фирмы. Формат отчета создается ролью, такой как руководитель, затем формат отчета сохраняется в ClearQuest Workspace. Для создания отчета выберите формат отчета, затем укажите результаты каких запросов будут вставлены в формат отчета. Отчеты хранятся в рабочей области и доступны для просмотра в любое время.

Этапы работы с инструментами

  • Создайте диаграмму
  • Создайте отчет

1. Создайте диаграмму

См. электронную справку ClearQuest: Справка > Оглавление и индекс > Работа с диаграммами > Создание диаграмм.

2. Создайте отчет

См. электронную справку ClearQuest: Справка > Оглавление и индекс > Работа с отчетами > Создание отчетов.

Также можно создать отчет для текущего активного запроса.  

См. электронную справку ClearQuest: Справка > Оглавление и индекс > Работа с отчетами > Создание отчетов > Создание отчетов в ClearQuest Windows из результатов запроса.
