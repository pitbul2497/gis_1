Руководство по инструменту: Поиск по web-сайту RUP
В этой памятке по инструменту описывается поиск по web-сайту Rational Unified Process.
Инструмент: Rational Unified Process
Основное описание

Обзор

При поиске рекомендации по процессу для определенной задачи или по применению инструмента в контексте проекта, зачастую существующие представления не позволяют вам легко найти искомое. Функция поиска RUP использует мощную службу, возвращающую отсортированные результаты, исходя из близости искомых терминов в теле, появления их в заголовке и частоты их упоминания в тексте.  Это позволяет быстро находить нужные материалы.   

В этой памятке по инструменту выполняются следующие шаги:

  • Запустить поиск
  • Ввести термины поиска
  • Выполнить поиск
  • Выбрать результат поиска
  • Дополнительная информация

Запустить поиск

Запустите поисковую службу, нажав кнопку Поиск в правой части серой панели над страницей содержимого RUP.  Появится диалоговое окно описка.

Ввести термины поиска

Введите искомые термины в четырех полях, отображаемых в правой части.  Доступны следующие варианты:

  • Любое слово: служба поиска выполнит поиск всех страниц, содержащих любой из введенных терминов.
  • Все слова: служба поиска выполнит поиск всех страниц, содержащих все введенные термины.
  • Точная фраза: служба поиска выполнит поиск всех страниц, содержащих всю фразу целиком.
  • Без слов: этот поиск работает совместно с терминами поиска, введенными в одном из трех полей.  Поиск просмотрит возвращенный набор результатов и отбросит материалы, содержащие термины из этого поля.

Вы можете выполнить поиск по конфигурации процесса RUP или по Web-сайту developerWorks®: Rational®. По умолчанию выполняется поиск по RUP.

Также на странице результатов можно отображать как больше 10 (используется по умолчанию), так и меньше 10 найденных станиц.

Выполнить поиск

Нажмите кнопку 'Начать поиск' в правой части окна. Результаты поиска будут показаны в диалоговом окне под полями с терминами поиска. На одной странице будет показано выбранное вами число результатов (по умолчанию 10). Вы можете перейти к следующему набору результатов.

При выборе в качестве искомого раздела developerWorks: Rational, откроется новое окно браузера, содержащее developerWorks: Rational.

Выбрать результат поиска

Выберите результат поиска, наиболее подходящий для вас. При поиске по RUP страница с этим результатом откроется в браузере RUP. Выберите браузер RUP для просмотра страницы.

При поиске по developerWorks: Rational страница с результатами откроется в том же браузере, что и результаты поиска.
