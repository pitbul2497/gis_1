Основное описание

Обзор

Эта памятка по инструменту применима к Microsoft Windows 98/2000/NT 4.0.

Для применения Rational Robot для настройки среды для тестирования выполните следующие действия:

  1. Настройте тестовую среду для записи, изменения или воспроизведения тестовых сценариев
  2. Настройте опции сценария записи GUI
  3. Задайте опции воспроизведения сценария GUI

1. Настройте тестовую среду для записи, изменения или воспроизведения тестового сценария

На воспроизведения сценария может повлиять состояние среды Microsoft Windows и тестируемого приложения. При наличии различий между средой записи и воспроизведения, могут возникнуть неполадки.

Перед записью или изменений тестового сценария, убедитесь, что тестируемое приложение и прочее ПО (в тестовой среде) находятся в соответствующем исходном состоянии. Кроме того перед выполнением тестов (воспроизведением тестовых сценариев) убедитесь, что ваши тестируемые приложения и прочее ПО находятся в том же исходном состоянии, что и при записи/изменении тестовых сценариев. Все приложения и окна, которые были открыты, активны или отображались при записи сценария, должны быть также открыты, активны и отображаться при воспроизведении. Кроме того, убедитесь, что все относящиеся к делу сетевые параметры, активные базы данных и системная память находятся в том же состоянии, что и при записи сценария.

При воспроизведении тестовых сценариев убедитесь, что вы задали соответствующие опции воспроизведения, включая способ обработки Rational Robot непредвиденных активных окон и восстановление после ошибок команд сценариев (см. следующий раздел).

2. Настройка опций записи сценария GUI

Опции записи сценария графического пользовательского интерфейса (GUI) предоставляют Rational Robot инструкции, касающиеся обработки определенных объектов, перемещений их мышкой, свойств окон, содержимого объектов и окна Robot при записи сценария.

Чтобы задать опции записи сценария GUI выполните следующие действия:

  1. Откройте окно Опции записи GUI одним из следующих способов:
  • Перед началом записи выберите Инструменты > Опции записи GUI.
  • Начните запись, нажав кнопку Записать сценарий GUI. В окне Запись выберите Опции.
  1. Задайте опции на всех вкладках. Дополнительные сведения о каждой вкладке приведены в главе Настройка опций записи GUI в руководстве Применение Rational Robot или см. электронную справку Robot.

Общие - указывает как Robot определяет/распознает содержимое списков и меню, а также неподдерживаемые перемещения объектов мышью. Кроме того, на этой вкладке можно также указать приставку для автоматического именования сценариев, выбрать, должен ли Robot сохранять и восстанавливать размеры и расположение активных окон, записывать время обдумывания и записывать задержку после нажатия клавиши Enter.

Окно Robot - укажите как будет выглядеть окно Robot во время записи тестового сценария. Эта вкладка также позволяет пользователю указать клавиши быстрого доступа для отображения и скрытия окна Robot и перехода между объектно- ориентированной записью и низкоуровневой записью.

Порядок распознавания объектов - используется для изменения приоритета методов распознавания объектов для отдельных типов объектов до или после записи.

Сведения о записи сценария приведены в Памятке по инструменту: создание тестовых сценариев с помощью Rational Robot.

3. Задайте опции воспроизведения сценария GUI

Опции воспроизведения сценариев GUI задают инструкции для Rational Robot относительно воспроизведения сценариев GUI. Задать эти опции можно до начала воспроизведения или в начале процесса воспроизведения.

Для того чтобы задать опции воспроизведения GUI выполните следующие действия:

  1. Откройте окно Опции воспроизведения GUI одним из следующих способов:
  • Перед началом воспроизведения выберите Инструменты > Опции воспроизведения GUI.
  • Начните воспроизведения, нажав кнопку Воспроизвести сценарий на панели инструментов. В окне Воспроизведение выберите Опции.
  1. Задайте опции на всех вкладках.  

    Дополнительные сведения о каждой вкладке приведены в главе Настройка опций воспроизведения GUI в руководстве Применение Rational Robot или см. электронную справку Robot.

  2. Воспроизведение - задайте задержку между командами и нажатиями клавиш, укажите, следует ли применять записанные время обдумывания и задержки ввода, пропускать точки проверки, отображать окно с подтверждением результата и что произойдет с окном Воспроизведение при воспроизведении. Дополнительные сведения можно получить нажав кнопку Справка.

    Протокол - укажите какие результаты следует сохранять в протоколе, появляется ли окно протокола автоматически после воспроизведения, следует ли отображать подтверждение перезаписи протокола и использовать сведения протокола по умолчанию или предоставляемые вами.

    Совпадение названий - укажите как Robot сопоставляет названия окон, захваченные при записи, с названиями, обнаруженными при воспроизведении. Дополнительные сведения можно получить нажав кнопку Справка.

    Состояние ожидания - укажите значения повтора попыток и тайм-аута по умолчанию для воспроизведения.

    Непредвиденные активные окна - укажите как Robot реагирует на появление непредвиденных активных окон.

    Восстановление после ошибок - укажите как Robot выполняет восстановление после возникновения ошибок сценария и сбоев точек проверки при воспроизведении сценариев.

    Прерывание - укажите захватываемые утилитой Прерывание сведения о сбоях общей защиты при воспроизведении и восстановлении.

  3. Нажмите кнопку OK.

Сведения о воспроизведении сценария приведены в Памятке по инструменту: создание тестовых сценариев с помощью Rational Robot.
