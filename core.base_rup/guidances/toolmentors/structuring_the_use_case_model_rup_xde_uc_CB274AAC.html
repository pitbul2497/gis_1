Обзор 

Шаги, описанные в этой памятке по инструменту, соответствуют этапам выполнения задачи. Ссылки на разделы электронной справки Rational XDE™ помечены .

В этой памятке предполагается, что модель варианта, заполненная субъектами и вариантами, уже была создана согласно инструкциям, описанным в Памятке по инструменту: поиск субъектов и вариантов с помощью Rational XDE.

Эта памятка по инструменту состоит из следующих действий:

  • Идентификация общих требований
  • Установка взаимоотношений добавления между вариантами
  • Установка взаимоотношений расширения между вариантами
  • Установка обобщений между вариантами
  • Установка обобщений между субъектами
  • Вычисление результатов

Идентификация общих требований 

Общее использование вариантов достигается за счет добавления взаимоотношений между существующими вариантами или создания новых вариантов.Для этого выполните следующие действия:

  1. Откройте модель прецедентов.
  2. Перейдите к пакету, содержащему варианты. См. Rational XDE™ Рекомендации по структуре модели.
  3. В Памятке по инструменту: поиск субъектов и вариантов с помощью Rational XDE создавалась диаграмма вариантов, содержащая все варианты и субъекты.Перейдите к диаграмме.
  4. Выполните рефакторинг для варианта, создав при необходимости новый. Процедура создания нового варианта с помощью Rational XDE описана в Памятке по инструменту: поиск субъектов и вариантов с помощью Rational XDE. Создание подробного описания этих новых вариантов описано в Памятке по инструменту: создание подробного описания варианта с помощью Rational XDE.
  5. Добавьте взаимоотношения между вариантами. В следующих разделах описывается добавление всех типов взаимоотношений.
  6. Опишите все взаимоотношения. См. Документирование элементов модели .

Создайте дополнительные диаграммы прецедентов и запакуйте их при необходимости, чтобы упростить управление. Справочник: Добавление диаграмм Группировка элементов модели в пакетах .

Установка обобщений между субъектами

  1. Откройте модель прецедентов.
  2. Перейдите к пакету, содержащему субъекты. См. Рекомендации по структуре модели Rational XDE.
  3. В Памятке по инструменту: поиск субъектов и вариантов с помощью Rational XDE создается диаграмма для захвата всех субъектов.Откройте эту диаграмму.
  4. Выполните рефакторинг существующих субъектов и создайте новые при необходимости. Процедура создания субъектов описана в Памятке по инструменту: поиск субъектов и вариантов с помощью Rational XDE.
  5. Добавьте обобщения между субъектами. См. Добавление взаимосвязей обобщений .
  6. При необходимости опишите все взаимосвязи обобщения. См. Документирование элементов модели .

Дополнительная информация приведена в разделе Работа с диаграммами прецедентов .
