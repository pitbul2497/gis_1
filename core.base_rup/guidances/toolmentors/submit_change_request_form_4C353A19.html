Руководство по инструменту: Отправка запросов изменений с помощью Rational ClearQuest
В этой памятке по инструменту описывается отправка запроса изменения или дефекта с помощью Rational ClearQuest.
Инструмент: Rational ClearQuest
Взаимосвязи
Связанные элементы
  • Анализ ошибки при тестировании
  • Обновите запрос изменения
  • Определение результатов тестирования
  • Оформление запроса изменения
  • Подтверждение повторного или отклоненного запроса изменения
  • Реализация процесса управления изменениями
Основное описание

Обзор

ClearQuest хранит запросы изменения в записях базы данных. Администратор ClearQuest может создать различные типы записей для разных целей и проектов. Каждый тип записи может иметь уникальные поля и требования к данным.

ClearQuest облегчает для пользователей отправку, изменения, отслеживание и создание диаграмм для запросов изменений при их перемещении по системе управления запросами изменений.

Администратор ClearQuest может создать пользовательский набор типов записи. Вы можете использовать один тип записи, такой как дефект, для всех запросов изменений, либо можно использовать различные типы записей для разных целей, таких как запросы расширения, документации и прочее.

Этапы работы с инструментами

См. электронную справку ClearQuest: Справка > Содержимое и индекс > Работа с записями > Отправка записей.
