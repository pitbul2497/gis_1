Руководство по инструменту: Применение наборов изменений UCM в Rational ClearCase
В этой памятке по инструменту описывается использование наборов изменений в операциях Unified Change Management (UCM) для создания и записи изменений.
Инструмент: Rational ClearCase
Взаимосвязи
Связанные элементы
  • Внесение изменений
Основное описание

Обзор

На следующей диаграмме показан поток операций UCM. Затемненные области описаны в этой памятке.

Поток операций UCM

Примечание о терминологии: Операция ClearCase UCM схожа с понятием Порядок работы в RUP.  Не стоит путать ее с концепцией RUP для операции.

Операция UCM отслеживает версии, созданные при работе над задачей разработки. Операция включает в себя текстовый заголовок, описывающий ее, ИД пользователя создателя операции и набор изменений, идентифицирующий все созданные версии при работе с операцией.  Когда вы готовы к изменению исходных файлов, необходимо задать представление разработки для операции.

Необходимо задать операцию для представления разработки перед тем, как можно будет изменить файлы.  

Эта памятка по инструменту применима к Microsoft Windows.  

Этапы работы с инструментами

Для работы с операциями UCM:

  1. Создайте или найдите набор операций
  2. Изымите и измените версии
  3. Проверьте работу

1. Создайте или найдите и задайте набор операций 

При изъятии файлов или каталогов в проекте UCM потребуется указать операцию для отслеживания работы.

Создайте новую операцию и задайте ее в текущем представлении.

  1. В окне Изъять, Добавить или Добавить в управление исходным кодом нажмите кнопку Создать.
  2. В окне Создать операцию введите заголовок операции.
  3. Нажмите кнопку OK.
  4. Для добавления этой операции к потоку и ее указания в представлении, нажмите кнопку OK.

Найдите и задайте операцию.

Операции обслуживаются между рабочими сеансами. Выполните эти действия для поиска существующей операции и ее указания в представлении разработки.  

Из обозревателя ClearCase:

  1. В обозревателе ClearCase перейдите на вкладку Представления.  
  2. Щелкните на странице проекта и выберите представление разработки.  
  3. На панели Папка выберите Мои операции.
  4. На панели Сведения выберите переключатель рядом с задаваемой операцией.

В окнах ClearCase:

  1. В окне Изъять, Добавить или Добавить в управление исходным кодом выполните следующие действия:
  • Выберите операцию в списке Операция.
  • Нажмите кнопку Обзор и укажите критерий выбора.
  1. Нажмите кнопку OK. ClearCase отобразит в представлении операцию, выбранную в списке Операция.

2. Изъятие и изменение версий 

Перед изменением исходных файлов перейдите к представлению Разработка и изымите их. Изъятие делает версии файлов или каталогов доступными для записи в представлении.  

  1. В обозревателе ClearCase выберите файлы или каталоги, затем щелкните правой кнопкой мыши на них и выберите Изъять.  
  2. ClearCase откроет окно Изъятие с выбранной текущей операцией в списке Операция. Вы можете принять этот выбор, выбрать другую операцию или создать новую.
  3. В поле Комментарии опишите планируемые изменения. В браузере свойств версии и в браузере хронологии элемента появятся комментарии.
  4. Выполните одно из следующих действий:
  • Для изъятия только одного файла нажмите кнопку OK.
  • Для применения текущих параметров для всех выбранных элементов нажмите кнопку Применить ко всем.
  • Чтобы прервать процесс изъятия и оставить данный и оставшиеся элементы добавленными, нажмите кнопку Отмена.

3. Добавление работы 

Для сохранения записи текущего состояния файла добавьте его. При добавлении файлов или каталогов в VOB добавляются новые версии. Сведения о версиях записываются текущей заданной операцией.   

После добавления текущая операция остается в представлении.  

  1. В обозревателе ClearCase выберите изъятые файлы или каталоги.  
  2. Щелкните правой кнопкой на выбранных элементах и нажмите кнопку Добавить.
  3. В поле Комментарии можно переписать или добавить текст к комментарию, введенному при изъятии папки или файла.
  4. Выполните одно из следующих действий:
  • Для добавления только одного файла нажмите кнопку OK.
  • Для применения текущих параметров для всех выбранных элементов нажмите кнопку Применить ко всем.
  • Чтобы прервать процесс добавления и оставить данный и оставшиеся элементы изъятыми, нажмите кнопку Отмена.
