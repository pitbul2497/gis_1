Руководство по инструменту: Работа с диаграммами с помощью Rational ClearQuest
В этой памятке по инструменту описывается создание и применение диаграмм для анализа результатов запроса с помощью RationalClearQuest.
Инструмент: Rational ClearQuest
Взаимосвязи
Связанные элементы
  • Создание отчета о состоянии конфигурации
Основное описание
Обратите внимание, что функции диаграммы Rational ClearQuest доступны только в Windows.

Обзор

Диаграммы являются графическим способом отображения сведений, обнаруженных запросом. Диаграммы являются способом графического запроса данных для дальнейшего анализа. Rational ClearQuest предоставляет три типа диаграмм для анализа данных: распределение, тенденция и давности. Можно отобразить диаграмму в пяти форматах: круговая, столбиковая, со вкладками, диаграмма областей и линейная.

Диаграмма состоит из трех основных частей: оси x для группировки данных и оси y для определения способа измерения данных, а также из области условных обозначений.

В этой памятке описываются действия для:

  • Создания и сохранения диаграммы
  • Выполнения диаграммы
  • Печати диаграммы

Создания и сохранения диаграммы

Для создания диаграммы распределения, тенденции или давности используйте мастер диаграмм. Для создания диаграммы выполните следующие действия:

  1. Выберите Запрос > Создать диаграмму или нажмите кнопку Диаграмма на панели инструментов.
  2. Выберите тип записи для диаграммы и нажмите кнопку OK.
  3. В разделе Тип диаграммы выберите тип создаваемой диаграммы и нажмите кнопку Далее.
  4. Выберите переключатель Выполнить запрос для запуска запроса при создании диаграммы. Если оставить переключатель Выполнить запрос отключенным, результаты диаграммы будут пустыми пока вы не укажите записи, которые следует добавить в диаграмму, на вкладке Редактор запросов.
  5. Укажите параметры для типа диаграммы и нажмите кнопку Далее.
  6. Задайте метки для диаграммы и нажмите кнопку Далее.
  7. Выберите тип отображения для диаграммы и нажмите кнопку Далее. Для диаграмм распределения удобно использовать столбиковые, круговые и диаграммы со вкладками. Линейные и диаграммы областей удобно применять для диаграмм тенденций и давности.
  8. Настройте диаграммы, выбрав переключатели для стилей, которые следует добавить к диаграмме.
  9. Нажмите кнопку Готово для создания диаграммы.
  10. Для сохранения диаграммы выберите Файл > Сохранить или Файл > Сохранить как.

Выполнения диаграммы

Сохраненные диаграммы можно выполнить при необходимости проанализировать данные. Для запуска диаграммы выполните следующие действия:

  1. Дважды щелкните на диаграмме в рабочей области.
  2. В зависимости от выбранной диаграммы может потребоваться ввести параметры, уточняющие запрос, связанный с диаграммой. В таком случае укажите параметры и нажмите OK или Выполнить запрос.

Печати диаграммы

Для печати диаграммы выполните следующие действия:

  1. Щелкните на диаграмме, которую следует распечатать.
    • Для печати отображение области диаграммы должно быть активно.
    • Для предварительного просмотра печати выберите Файл > Просмотр печати.
    • Для печати диаграммы в черно-белом варианте следует просматривать ее в черно-белом. Убедитесь, что опция Просмотр > Цвет не выбрана.
  3. Выберите Файл > Печать диаграммы.
  4. В диалоговом окне Печать выберите нужные опции и нажмите кнопку OK.
