В этом руководстве приведены инструкции по извлечению информации из базы данных ClearQuest путем оформления запросов с помощью Rational ClearQuest.
Инструмент: Rational ClearQuest

Основное описание

Обзор

В этом руководстве описаны действия для:

  • Оформления и сохранения запросов
  • Просмотра результатов запросов
  • Вывода результатов запросов
  • Печати запросов

Оформление и сохранение запросов

Существует множество способов оформления запросов, однако в каждом случае запрос необходимо назвать, определить, выполнить и/или сохранить. При первом использовании базы данных следуйте инструкциям мастера запросов, чтобы выбрать отображаемые поля, а также поля и соответствующие значения для настройки фильтрации запросов. Когда вы познакомитесь с процедурой создания запросов, редактировать фильтры и задавать отображаемые поля можно будет в окне Запрос.

Этапы работы

Для оформления запроса с помощью Rational ClearQuest и мастера запросов выполните следующие шаги:

  1. В меню Запрос включите переключатель для опции Активировать мастер запросов.
  2. Выберите Запрос > Оформить запрос. Откроется окно Выбрать тип данных.
  3. Выберите тип данных и нажмите OK.
  4. В окне Мастер запросов ClearQuest выполните следующие действия:
    1. В разделе Начать с существующего запроса введите имя запроса, который будет служить шаблоном.
    2. Для просмотра списка запросов и выбора запроса, который будет служить шаблоном, нажмите Обзор и затем Да. Рекомендуется выбрать запрос из группы Общие запросы.
    3. Для создания пустого запроса с помощью мастера запросов нажмите Далее.
  5. В окне Определить внешний вид запроса выберите поля, которые будут показаны в результатах запроса. Например, если выбрать поля ИД, Состояние и Заголовок, в результатах запроса будут показаны только эти три поля.
    1. Дважды щелкните на каждом поле, которое будет показано в результатах запроса. Вобранные поля будут перенесены в таблицу Внешний вид. Таблица Внешний вид содержит следующие столбцы: Поле, Название, Сортировать, Показать и Порядок сортировки.
    2. Выберите нужные поля и нажмите Далее.
  6. Выберите поля, которые будут служить фильтрами запросов. Фильтры определяют данные, которые будут показаны в результатах запроса.
    1. В окне Выбрать поля для фильтрации запроса выберите поля, которые будут служить фильтрами запроса. Дважды щелкните на поле, чтобы переместить его в панель Фильтры.
    2. Нажмите кнопку Далее.
  7. На странице мастера запросов Настройка фильтров задайте выражения, которые будут искать указанные действительные значения или диапазон значений. Страница Настройка фильтров разделена на две части: панель Фильтры (слева), где перечислены выбранные поля, и панель Значения (справа), где можно задать значения фильтров.
    1. Выберите фильтр, для которого нужно задать значение.
    2. В панели Значения (справа) выберите оператор. Список доступных операторов зависит от типа данных для данного поля. Для выполнения оператора, обратного к выбранному оператору, включите переключатель Обратный.
    3. Для просмотра списка значений, доступных для данного фильтра, выберите Значения.
    4. В окне Значения выберите нужные значения и нажмите OK.
    5. Выберите опцию Задать значение и задайте значение для выбранного поля. Повторите этот шаг для каждого поля.
  9. Для завершения процедуры нажмите Выполнить. В результате запросу будет присвоено имя по умолчанию, и он будет выполнен. Появится окно Запрос, где будут указаны результаты.
  10. Если вы удовлетворены результатами и хотите выполнить запрос повторно, сохраните его. Для этого выберите Файл > Сохранить или Файл > Сохранить как. Укажите имя и расположение запроса и сохраните его.
  11. Для того чтобы аннулировать запрос и закрыть окно Запрос, нажмите Файл > Закрыть и затем Нет.

Просмотр результатов запроса

Результаты запроса перечислены на вкладке Результаты в таблице, где указаны те поля, которые вы выбрали при настройке внешнего вида запроса. Данные распределены по вкладкам таблицы Результаты, например, по вкладкам Основные, Анализ, Детализация. В зависимости от настройки ClearQuest поля на вкладах и количество вкладок для каждой записи будут разными.

Этапы работы

Для просмотра результатов запроса с помощью Rational ClearQuest выполните следующие действия:

  1. Щелкните на строке, где указана запись из базы данных.
  2. Для получения более подробной информации об этой записи перейдите на соответствующую ссылку.

Вывод результатов запроса

Rational ClearQuest сохраняет запросы в одной из двух папок: Частные запросы или Общие запросы. Создаваемые запросы автоматически записываются в папку Частные запросы. Запросы из обеих папок готовы для выполнения. При необходимости их можно настроить для конкретной ситуации.

Этапы работы

Для вывода результатов запроса с помощью Rational ClearQuest выполните следующие действия:

  1. Откройте одну из папок (Частные или Общие запросы).
  2. Дважды щелкните на запросе, который нужно выполнить.
  3. При необходимости задайте параметры в появившемся окне и нажмите OK.

Печать результатов запросов

После выполнения запроса можно напечатать данные, указанные на вкладке Результаты. При необходимости можно добавить верхние и нижние колонтитулы. Формат отчета определяется типом данных и от него зависит, какие поля выводятся в печать.

Этапы работы

Для печати результатов запроса с помощью Rational ClearQuest выполните следующие шаги:

  1. Выполните запрос и перейдите на вкладку Результаты.
  2. В таблицы можно добавить верхние и нижние колонтитулы, которые будут выводиться при печати. Для изменения колонтитулов выполните следующие действия:
    1. Выберите Файл > Колонтитулы.
    2. Перейдите на вкладку Верхние колонтитулы и добавьте или отредактируйте верхний колонтитул. Для этого введите текст, который будет показан в соответствующем столбце. Выберите выравнивание по центру, левому или правому краю.
    3. Перейдите на вкладку Нижние колонтитулы и отредактируйте нижний колонтитул.
    4. Для использования этих же параметров при следующем сеансе работы с Rational ClearQuest, выберите опцию Сохранить параметры в профайл.
    5. Нажмите OK.
  3. Нажмите Файл > Печать таблицы или нажмите кнопку Печать на панели инструментов.
  4. В окне Печать задайте нужные значения для опций Принтер, Диапазон страниц и Число копий и при необходимости включите переключатель для опции Печать в файл. Нажмите OK.
