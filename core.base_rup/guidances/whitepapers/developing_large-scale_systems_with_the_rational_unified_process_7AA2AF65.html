Аннотация

При создании крупномасштабных систем процесс разработки значительно усложняется. Увеличивается не только число рабочих продуктов, но и возрастает объем ресурсов, которыми нужно управлять. В этом документе приведено описание шаблона архитектуры, который позволяет контролировать сложную систему. Такой шаблон архитектуры называется системой взаимосвязанных систем.

В этом документе приведено описание шаблона архитектуры для систем взаимосвязанных систем. Данная архитектура позволяет выполнять рекурсию не только в рамках одной модели. Она рассматривает каждую подсистему как самостоятельную единицу и позволяет выполнять рекурсию между всеми наборами рабочих продуктов каждой системы. Эту архитектуру можно использовать для систем, состоящих из нескольких взаимодействующих систем. Каждая из составляющих систем содержит собственный набор моделей, независимых от моделей других систем.

Примеры из этого бюллетеня свидетельствуют, что архитектуру для моделирования систем взаимосвязанных систем удобно применять в разных прикладных областях. По сути, ее можно использовать для любой системы, части которой можно рассматривать как отдельные системы.
