Аннотация

Можно произвести оценки объема работ по разработке на основании параметров модели вариантов использования, в которой зафиксированы функциональные требования. Должен ли существовать основанный на варианте использования аналог функциональных требований?

Этот официальный документ рассматривает некоторые важные приемы оценки процесса разработки, которые могут быть включены в RUP. Здесь представлена структура для оценки на основании вариантов использования. Эта структура рассматривает идею уровня, размера и сложности варианта использования для различных категорий систем.
