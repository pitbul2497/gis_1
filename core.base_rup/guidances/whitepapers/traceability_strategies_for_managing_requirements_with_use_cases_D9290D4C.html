Аннотация

Во многих коммерческих приложениях, использующих приемы моделирования вариантов использования, модель вариантов использования должна комбинироваться с более традиционными техниками фиксации требований, чтобы обеспечить процесс управления требованиями, приемлемый для всех заинтересованных лиц проекта.  Эта статья исследует стратегии трассируемости, доступные для организаций, использующих приемы моделирования вариантов использования в своей стратегии управления требованиями.
