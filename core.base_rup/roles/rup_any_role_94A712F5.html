    Квалификация

    Роль Любая роль может быть передана любому участнику проекта, который выполняет как минимум одну из конкретных ролей, определенных в RUP. Таким образом, обязательным навыком для выполнения этой роли является способность выполнять функции какой-либо другой роли в RUP; соответствующие навыки будут использоваться для выполнения задач, возложенных на роль Любая роль.

    Подходы к назначению

    Роль Любая роль выполняется всеми участниками проекта. Если участнику проекта назначена какая-либо другая роль, данная роль будет присвоена ему по умолчанию.
