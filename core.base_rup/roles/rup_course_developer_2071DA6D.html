  • Учебные материалы
    Основное описание

    Исполнитель роли Разработчик курса должен обладать опытом в создании как минимум вводного учебного курса к какому-либо продукту. Исполнитель этой роли должен хорошо разбираться в продукте, для которого создается учебный курс, и, желательно, понимать, как выглядит этот продукт с точки зрения конечного пользователя.

    Учебные материалы обычно включают слайды, демонстрируемые инструктором, примечания для слушателей курса, примеры, учебники и прочие материалы, позволяющие пользователям лучше понять работу продукта.

    Персонал
    Квалификация

    Исполнитель роли Разработчик курса должен обладать опытом в создании курса на основе выбранной в проекте технологии (например, самообучение через Web с инструктором). Желательны также навыки преподавательской деятельности. Исполнитель этой роли должен хорошо понимать принципы обучения и различные методики преподавания, что необходимо для составления эффективного курса.

    Подходы к назначению

    Назначить сотрудника на роль Разработчик курса можно одним из следующих способов:

    • Передать роль Разработчик курса одному или нескольким сотрудникам, которые также будут исполнять роль Аналитик. Это наиболее стандартный подход. В этом случае используются коммуникативные способности, которыми должен обладать Аналитик, и опыт, который он получает в ходе выполнения фаз Начальный этап и Уточнение.
    • Передать роль Разработчик курса одному или нескольким сотрудникам, которые будут исполнять только ее. Это хорошее решение для больших коллективов, особенно в том случае, если есть возможность привлечь к написанию (и, возможно, преподаванию) курса специализированных экспертов.
    • Нанять специалиста в области разработки учебных курсов. Это стандартный способ решения проблемы.

    Второй подход также можно использовать в том случае, если требуется уделить особое внимание области, к которой относится продукт; если эту роль выполняют эксперты - представители заинтересованного лица, вероятность создать пригодный курс повышается.
