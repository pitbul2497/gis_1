    Основное описание

    Координатор проверки управляет процессом проверки. Проверки можно разделить на две основные категории: проверки хода работы со стороны администрации и технические проверки рабочих продуктов. Задачи исполнителя данной роли состоят в следующем:

    • Контроль выполняемости всех необходимых проверок
    • Уведомление всех лиц, чье присутствие необходимо, и информирование их о деталях проверки
    • Контроль эффективности проверки и ее соответствия стандартам
    • Контроль выполнения заданий, которые были признаны необходимыми по результатам проверки

    Присваивая сотруднику эту роль, необходимо учесть как способности, необходимые для ее выполнения, так и выбранную общую стратегию присвоения ролей.

    Персонал
    Квалификация

    Исполнитель роли Координатор проверки должен быть способен обеспечить эффективное взаимодействие всех групп проверяющих. В идеале исполнитель этой роли должен пользоваться доверием и уважением со стороны каждого участника проверки.

    Способности и знания, необходимые для выполнения этой роли:

    • Организационные навыки и навыки планирования
    • Дипломатичность и способность к разрешению споров
    • Навыки оказания содействия
    • Способность обеспечить продуктивное сотрудничество
    Подходы к назначению

    Исполнитель роли может быть назначен одним из следующих способов:

    • Решение о назначении сотрудника на роль Координатор проверки принимается, исходя из особенностей проекта. Это стандартный подход для небольших коллективов, особенно в том случае, если между членами коллектива установлены доверительные и уважительные отношения. Он особенно эффективен в том случае, если коллектив состоит из опытных сотрудников с приблизительно одинаковыми способностями.
    • В небольших и средних коллективах роль Координатор проверки может исполнять Руководитель проекта. В некоторых случаях эту роль может исполнять лидер коллектива.
    • В больших командах сотрудники, выполняющие административные задачи, могут выступать в качестве кураторов, допуская экспертов к активному участию в проверке.
