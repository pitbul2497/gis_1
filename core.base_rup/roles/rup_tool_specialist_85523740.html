Роль: Специалист по инструментам
Исполнитель этой роли обеспечивает работу инструментальных средств, используемых в ходе проекта. Сюда входят выбор и получение инструментов,их настройка и установка, а также проверка их функциональности.

    Исполнитель роли Специалист по инструментам должен обладать широким набором навыков. Сюда входит глубокое понимание работы проекта. Получить достаточные знания по этому вопросу необходимо перед началом проекта. Для сравнения и выбора инструментов требуются базовые навыки систематического анализа. Обязательно знание платформы разработки, в особенности сетевых аспектов. Исполнитель ролиСпециалист по инструментам должен обладать хорошими коммуникативными навыками и способностью осуществлять техническую поддержку, так как при установке инструментов и устранении неполадок, возникших при работе с ними, участники проекта скорее всего будут обращаться к нему.

    Подходы к назначению

    Назначить сотрудника на роль Специалист по инструментам можно одним из следующих способов:

    • Передать роль Специалист по инструментам одному или нескольким сотрудникам, которые также будут исполнять роль Реализатор. Это стандартный подход, особенно в небольших и средних коллективах разработки. При этом учитывается тот факт, что навыки разработчика являются необходимыми для исполнителей обеих ролей.
    • Передать роль Специалист по инструментам одному сотруднику, который будет исполнять только ее. Это еще один стандартный подход, используемый в больших и средних коллективах в том случае, если ключевым аспектом плана разработки является эффективная работа инструментов и автоматизация процесса.
