Задача: Создание блока развертывания
Эта задача описывает создание носителя для решения, который позволит развернуть его в целевой среде.
Дисциплины: Управление конфигурацией и изменениями
Взаимосвязи
РолиОсновной исполнитель:
  • Администратор конфигурации
Дополнительные исполнители:
ВходыОбязательный:
  • Компиляция
  • План развертывания
    		Необязательный:
    • Артефакты установки
    • Материалы поддержки пользователей
    • Модель развертывания
    • Перечень файлов
    • Примечания к выпуску
    • Учебные материалы
    • Хранилище проектов
      Выходы
      • Блок развертывания
        Основное описание

        Все рабочие продукты проекта физически хранятся в хранилище проекта, а логически они организованы в соответствии со структурой каталогов продукта.   Модуль развертывания содержит все необходимые элементы, которые перечислены в Описи материалов.  

        В этой задаче Администратор конфигурации создает копию разворачиваемых элементов контрольной версии в хранилище проекта   на подходящем для развертывания в целевой среде носителе. Этим носителем может быть компакт-диск, или в случае загружаемого по сети продукта - архивированная копия, доступная для загрузки.
