Задача: Разработка материалов поддержки
Эта задача описывает разработку материалов поддержки пользователя.
Дисциплины: Развертывание
Взаимосвязи
РолиОсновной исполнитель:
  • Технический писатель
Дополнительные исполнители:
ВходыОбязательный:
  • Компиляция
  • План итерации
    		Необязательный:
    • Раскадровка
    • Руководство по стилю публикаций
    • Спецификация требований к программному обеспечению
    • Схема навигации
      Выходы
      • Материалы поддержки пользователей
        Основное описание

        Сегодня для наиболее профессионально спроектированных продуктов материалы поддержки конечного пользователя сами создаются в виде приложений, таких как справочные системы или Web-сайты. Создание таких приложений часто является само по себе отдельным проектом, который должен следовать за указанными ниже шагами в каждой итерации жизненного цикла [HAC97]:

        • Создать план высокого уровня для информации, которая должна быть представлена.
        • Указать подробное содержимое.
        • Реализовать информационное содержимое.
        • Протестировать и создать материалы.
        • Оценить.
