        Основное описание

        Разработайте новое или используйте уже существующее руководство по стилю публикаций. Это руководство поможет сохранить общий стиль для всех справочных материалов, издаваемых в рамках проекта (см. раздел Рабочий продукт: Справочные материалы).

        Попробуйте использовать уже существующее руководство по стилю публикаций. Примеры таких руководств: [HAC97] и [MOS98].

        Занесите решения, касающиеся стиля публикаций, в Рабочий продукт: Руководство по стилю публикаций.
