      Основное описание

      В задаче Начало итерации руководитель проекта выделяет персонал и другие ресурсы проекта для каждой деятельности, которая будет выполняться в данной итерации, как определено в текущем плане итерации.

      Шаги
      Назначение персонала для деятельностей

      Для каждой деятельности руководитель проекта назначает персонал, имеющий соответствующие навыки для выполнения данной работы. Следуйте общепринятой практике руководства проектом и сбалансируйте общую рабочую нагрузку для каждого назначенного сотрудника.

      Запрос и присвоение других ресурсов

      Для некоторых деятельностей потребуется применение других специализированных ресурсов (например, выделенное процессорное время на мейнфрейме, применение специализированного аппаратного обеспечения тестирования). Для каждой деятельности руководитель проекта должен обеспечить доступность необходимых ресурсов и запланировать их использование

      Создание нарядов на работу

      После выделения для деятельностей текущей итерации персонала и других ресурсов администратор проекта документирует назначения в ряде нарядов на работу. Наряды на работу затем выдаются коллективу проекта для выполнения.
