Задача: Создание отчета о текущем состоянии
В этой задаче рассмотрен процесс регулярного обновления текущего состояния проекта.
Дисциплины: Управление проектом
Назначение
  • Предоставление регулярных обновлений текущего состояния проекта контрольному совету проекта.
  • Передача неполадок, входящих за пределы компетенции руководителя проекта, на рассмотрение контрольному совету проекта.
Взаимосвязи
РолиОсновной исполнитель:
  • Руководитель проекта
Дополнительные исполнители:
ВходыОбязательный:
  • Нет
Необязательный:
  • Оценка состояния
  • Показатели проекта
  • Список дефектов
  • Список рисков
    Выходы
    • Оценка состояния
      Шаги
      Подготовка отчета о текущем состоянии
      Цель Предоставление текущего состояния проекта контрольному совету проекта.

      Инструкции по оценке текущего состояния проекта приведены в разделе Рабочий продукт: Отчет о текущем состоянии. В качестве исходных данных применяются Список неполадок и  Показатели проекта, описанные в разделе Задача: Мониторинг состояния проекта. Отчет о текущем состоянии должен содержать следующую информацию:

      Техническое состояние: Работа, выполненная в ходе отчетного периода (например, завершенные задачи и рабочие продукты). Особое внимание следует уделить отставанию.

      В текущей итерации в отчет следует включить возможные изменения рамок проекта и качества (с учетом обнаруженных дефектов, которые не будут исправлены), необходимые для запланированного завершения итерации (например, для обеспечения 'соответствия расписанию').

       
      Состояние бюджета: Расходование средств. Особое внимание следует уделить неплановым расходам.  
      Состояние выполнения относительно запланированных вех: Были ли достигнуты запланированные вехи?  
      Выполненная работа/рамки проекта: Обновленная оценка рамок проекта в соответствии с выполненной работой и оценка выполненной работы.  
      Персонал: Сведения о персонале. Укажите вопросы и замечания.  
      Риск: Состояние реализации рисков.  
      Возникающие неполадки: Неполадки, требующие внимания контрольного совета проекта. Укажите рекомендации относительно возможных способов устранения.  
      Вопросы, требующие внимания: Список важных вопросов из предыдущих оценок и их текущие состояния.  

      Отчет о текущем состоянии, как правило, публикуется для контрольного совета проекта и рассматривается им в ходе проверки.
