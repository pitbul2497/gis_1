Задача: Обновите рабочую область
Данная задача подчеркивает важность регулярного обновления рабочих областей.
Дисциплины: Управление конфигурацией и изменениями
Назначение
  • Сделать так, чтобы члены проектной группы всегда работали с последними версиями файлов проекта. Для этого в представлении (рабочей области) разработки каждого члена проектной группы должны быть показаны файлы из последней утвержденной контрольной версии.

      Основное описание

      Для любой системы управления конфигурацией очень важны две вещи: контроль версий всех рабочих продуктов и наличие доступа к одобренным контрольным версиям. Идея обновления представлений для того, чтобы членам проектной группы была доступна новейшая версия коллективной работы, лежит на поверхности. Данная задача должна быть простой, и выполнять ее нужно регулярно.

      Каждый член проектной группы после обновления представления получает доступ ко всем рабочим продуктам через стандартный интерфейс "извлечения, изменения, компоновки, тестирования и регистрации", описанный в разделе Задача: Внесение изменений.
