Основное описание

IBM® Rational® Administrator служит для централизации управления проектом Rational Administrator и управляет связью хранилищ данных продуктов Rational с пользователями и группами для Rational Test. Rational Administrator применяется для создания проекта, подключения к проекту, создания хранилища данных Test, для интеграции тестовых продуктов и Rational RequisitePro, создания интегрированной базы данных Rational ClearQuest и преобразования предыдущих версий хранилища для нового проекта.
