Инструмент: Rational Method Composer
Rational Method Composer является инструментом создания методов IBM Rational следующего поколения, обеспечивающим создание, управление и web-публикацию указаний по методам.
Взаимосвязи
Руководства по инструментам
  • Создание конфигурации метода с помощью Rational Method Composer
  • Создание встраиваемого модуля метода с помощью Rational Method Composer
  • Разработка процессов с помощью Rational Method Composer
  • Разработка содержимого метода с помощью Rational Method Composer
  • Навигация по содержимому метода с помощью Rational Method Composer
  • Экспорт процессов в инструмент планирования с помощью Rational Method Composer
  • Публикация конфигурации метода с помощью Rational Method Composer
Основное описание
Приложение IBM® Rational® Method Composer (RMC) является инструментом настройки и публикации процессов, который позволяет технологам процесса ускорить доставку настроенных процессов разработки программного обеспечения и использовать передовые методики, заложенные в приложении RUP®.   RMC оказывает помощью в настройке RUP в соответствии с требованиями данной организации, включая специальные знания, методики и внутренние знания компании.

Основанный на платформе Eclipse - Rational Method Composer (RMC) специально разработан для управления содержимым методов с такими функциями, как основанное на форме проекта создание, просмотр содержимого, поиск по содержимому и импорт и экспорт содержимого метода. RMC поддерживает детализированную настройку содержимого метода во время публикации, что позволяет удовлетворить разнообразные требования различных проектов и конечных пользователей. Функция RMC встраиваемого модуля содержимого допускает расширение и изменение существующего содержимого через встраиваемые модули, упрощая управление и обслуживание содержимого метода.
