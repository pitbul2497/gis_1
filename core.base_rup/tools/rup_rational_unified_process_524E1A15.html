Основное описание

IBM® Rational Unified Process® представляет собой гибкую платформу процесса разработки программного обеспечения. Благодаря настраиваемой архитектуре, платформа RUP® позволяет выбирать и развертывать только те компоненты процесса (включая роли, деятельности, шаблоны, рекомендации и памятки по инструментам), которые требуются на каждом этапе проекта. Имея в своей основе наилучшие проверенные на практике приемы разработки программного обеспечения, платформа RUP включает инструменты для настройки RUP в соответствии с потребностями данного проекта, инструменты для включения внутренних сведений в компоненты процесса, мощные настраиваемые web-инструменты развертывания, а также виртуальное сообщество для обмена опытом с коллегами и лидерами отрасли. Используя опробованную методологию и участвуя в общем комплексном процессе, члены вашего коллектива смогут взаимодействовать и выполнять свои функции более эффективно.

Браузер RUP является 'окном' в настройку пользовательского процесса, позволяя просматривать различные его представления, создавать собственные представления, выполнять поиск по содержимому и получать с помощью расширенной справки доступ к его элементам, наиболее соответствующим вашему расположению среди инструментов.
