Инструмент: Rational TestFactory
Инструмент IBM(R) Rational(R)TestFactory(R) автоматически генерирует комплексные сценарии тестирования.
Взаимосвязи
Руководства по инструментам
  • Структуризация реализации теста в Rational TestFactory
  • Реализация созданных тестовых сценариев с помощью Rational TestFactory
  • Выполнение набора тестирования с помощью Rational TestFactory
  • Оценка результатов выполнения тестового комплекта с помощью Rational TestFactory
  • Анализ сбоев втестов с помощью Rational TestManager и TestFactory
  • Применение Rational TestFactory для измерения и оценки охвата теста кода для тестовых сценариев Rational Robot
  • Вычисление тестового покрытия с помощью Rational TestFactory
  • Настройка тестовой среды в Rational TestFactory
