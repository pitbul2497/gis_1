Инструмент: Rational TestManager
Этот инструмент является краеугольным камнем инструментов тестирования Rational, управляя всеми деятельностями тестирования.
Взаимосвязи
Руководства по инструментам
  • Создание тестового плана с помощью Rational TestManager
  • Создание тестового сценария с помощью Rational TestManager
  • Создание тестовых комплектов с помощью Rational TestManager
  • Выполнение тестовых операций с помощью Rational TestManager
  • Реализация автоматического комплекта тестирования с помощью Rational TestManager
  • Выполнение набора тестирования с помощью Rational TestManager
Основное описание

IBM® Rational® TestManager является краеугольным камнем всех инструментов тестирования Rational, управляя всеми деятельностями тестирования.
