Артефакт: Субъект
Этот артефакт определяет согласованный набор ролей, которые пользователи системы могут играть при взаимодействии с ней. Экземпляр этого артефакта может проигрываться либо отдельным пользователем, либо внешней системой.
Типы рабочих продуктов: Элементы модели
Расширяет: Требование программного обеспечения
Назначение

Этот рабочий продукт используется различными заинтересованными лицами для различных целей:

  • Системные аналитики - для определения рамок системы.
  • Проектировщики пользовательского интерфейса - для фиксации характеристик субъектов-людей.
  • Авторы вариантов использования - для описания вариантов использования и их взаимодействия с субъектами.
  • Аналитики объектов - для реализации вариантов использования и их взаимодействия с субъектами.

Описание
Краткая схема

При создании  этого артефакта следует принимать во внимание его различные аспекты, включая следующее:

  • различные заинтересованные группы или лица, которые могут вносить требования
  • различные типы требований (категории, размеры), которые необходимо принять во внимание
Доводка
Опции представленияПредставление UML: Субъект

Субъект может иметь следующие свойства:

  • Имя: Имя субъекта.
  • Краткое описание: Краткое описание сферы ответственности субъекта, а также для чего данному субъекту требуется система. 
  • Характеристики: Для субъектов-людей: физическая среда субъекта, сколько пользователей представляет субъект, уровень профессиональных знаний субъекта, уровень владения компьютером, другие используемые субъектом приложения, а также общие характеристики, такие как пол, возраст, культурные корни и так далее. 
  • Взаимосвязи: Взаимосвязи, такие как обобщения субъекта и ассоциации общения, в которых участвует субъект. 
  • Диаграммы: Все локальные для субъекта диаграммы, такие как диаграммы вариантов использования, изображающие ассоциации общения субъекта с вариантами использования. 

Выберите, какие свойства и каким образом вы будете использовать. В частности, вам потребуется решить, на каком уровне детализации следует описать свойство "Характеристики".
