Артефакт: Результаты контроля конфигурации
Этот артефакт идентифицирует контрольную версию, все отсутствующие необходимые артефакты и не полностью протестированные или невыполненные требования.
Домены: Управление конфигурацией и изменениями
Типы рабочих продуктов: Оценка
Назначение
  • Для отчета о следующем:
    • производительность разработанного программного обеспечения соответствует требованиям
    • физически представлены необходимые рабочие продукты

  • Введение

  • Назначение контроля
  • Общая оценка
  • Контроль физической конфигурации
  • Идентификация контрольной версии
  • Отсутствующие рабочие продукты
  • Функциональный контроль конфигурации
  • Непротестированные требования
  • Неудовлетворенные требования
  • Открытые запросы на изменения
  • Корректирующие действия
  • Действие
  • Ответственный за выполнение
  • Контрольная дата
Доводка
Опции представления
