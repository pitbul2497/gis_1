Артефакт: Глоссарий
Этот артефакт определяет важные термины, используемые в проекте.
Домены: Требования
Типы рабочих продуктов: Спецификация
Назначение

Для системы существует один Глоссарий, предоставляющий согласованный набор определений, позволяющий избежать неправильного понимания. Участники проекта изначально используют этот артефакт для понимания терминов, специфических для данного проекта. Этот документ также важен для сотрудников, выполняющих следующие роли:

  • Разработчики, которые используют термины в Глоссарии при проектировании и реализации классов, таблиц баз данных, пользовательских интерфейсов и так далее
  • Аналитики, которые используют Глоссарий для фиксации относящихся к проекту терминов, так чтобы можно было четко определить правила бизнес-процесса и гарантировать, что в спецификациях требований правильно и согласованно применяются эти термины
  • Разработчики курса и редакторы технической документации, которые используют Глоссарий для создания обучающих материалов и документации с применением общепризнанной терминологии

Доводка
Опции представления

В некоторых проектах, в которых моделирование бизнес-процесса и моделирование домена не выполняется, Глоссарий является основным артефактом, который применяется для фиксации информации о домене бизнес-процесса проекта.
