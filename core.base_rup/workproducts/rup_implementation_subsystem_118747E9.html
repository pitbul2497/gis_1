Артефакт: Подсистема реализации
Этот артефакт содержит набор элементов реализации. Он выделяет структуру модели реализации, разделяя ее на более мелкие компоненты, допускающие отдельное интегрирование и тестирование.
Типы рабочих продуктов: Элементы модели
Назначение

Подсистема реализации используется следующими сотрудниками:

  • Архитекторами программного обеспечения - для разбиения структуры модели реализации на компоненты, которые можно интегрировать и протестировать по отдельности.
  • Проектировщиками следующей версии системы - для понимания структуры модели реализации.
  • Ответственными за реализации. других частей системы - для осознания того, как может использоваться создаваемая ими функциональная возможность.
  • Ответственными за тестирование подсистемы - для планирования деятельностей тестирования.
  • Руководителем проекта - в качестве основы для распределения работы по реализации.

Подсистема реализации является физическим аналогом пакета проектирования. Модель реализации и подсистемы реализации первоначально определяются в представлении реализации, и имеют важнейшее значение при выполнении реализации.
