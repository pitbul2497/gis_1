Артефакт: Артефакты установки
Этот артефакт состоит из программного обеспечения и задокументированных инструкций, необходимых для установки продукта.

Опции представления

Артефакты установки могут содержать следующие элементы:

  • Сценарии установки, файлы установки и так далее.
  • Инструкции по установке

Артефакты установки требуются, если для настройки системы в среде развертывания будут применяться программы установки. Если программное обеспечение будет развертываться только один раз (как в случае с несколькими системами, построенными компанией для внутреннего пользования на корпоративном сервере), артефакты установки можно опустить.

В системе, в которой предполагается установка продукта пользователем, инструкции по установке можно включить в руководство пользователя. В случае более сложной установки, при которой требуется квалифицированный персонал, требуется отдельное описание установки.
