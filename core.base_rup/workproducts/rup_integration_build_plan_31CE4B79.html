Артефакт: План компоновки интеграции
Этот артефакт представляет подробный план интеграции во время итерации.
Домены: Реализация
Типы рабочих продуктов: План
Назначение

Назначение плана компоновки интеграции - определить порядок, в котором следует реализовывать компоненты, какие компоновки создавать при интеграции системы, и как их следует оценивать.

План компоновки интеграции используется следующими сотрудниками:

  • Ответственными за реализацию - для планирования порядка реализации элементов проектирования и определения того, что и когда следует поставлять для интеграции системы
  • Ответственным за интеграцию - в качестве инструмента планирования
  • Разработчиком тестов - для определения тестов для данной итерации

Ключевые условия

Следует уточнить структуру плана компоновки интеграции в соответствии с особенностями данного проекта. Например, для системы большого размера целесообразно иметь вспомогательные планы для каждой подсистемы реализации. Формализм и рамки материалов теста, которые содержатся в плане компоновки интеграции либо на которые ссылается этот план, изменяются в зависимости от значимости компоновки. Очевидно, окончательная компоновка итерации будет официально оцениваться по отношению ко всем критериям, описанным в плане итерации.
