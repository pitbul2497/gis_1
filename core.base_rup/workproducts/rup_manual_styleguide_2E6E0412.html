Артефакт: Руководство по стилю публикаций
Этот артефакт описывает, как должны быть разработаны руководства для поддержки конечных пользователей.
Домены: Развертывание
Типы рабочих продуктов: Процесс
Назначение

Назначение руководства по стилю публикаций - описать стилистические соглашения, которые следует применять при разработке материалов для поддержки пользователей. Руководство по стилю публикаций содержит советы и правила по орфографии для общих и специальных терминов, стилю предложений и вопросам создания технической документации.

Доводка
Опции представления

Руководство по стилю публикаций требуется в том случае, если существует значительный объем документации для пользователей, включая и печатную, и электронную документацию.

Воспользуйтесь существующими руководствами по стилю, такими как [HAC97] и [MOS98], как полезными книгами, доступными по данной теме.
