Артефакт: План измерения показателей
Этот артефакт определяет задачи измерений, связанные показатели и простейшие показатели, которые должны быть собраны для мониторинга хода проекта.
Типы рабочих продуктов: План
Назначение

План измерения показателей программного обеспечения указывает, какие простейшие показатели следует собрать и какие показатели следует вычислить для мониторинга хода проекта по отношению к набору заданных целей проекта (см. раздел Концепции: Показатели). Он используется для сбора информации о проекте как входных данных для периодической оценки состояния (см. раздел Рабочий продукт: Оценка состояния).

Доводка
Опции представления

План измерения показателей физически может являться частью плана разработки программного обеспечения в случае наличия простой программы измерений.
