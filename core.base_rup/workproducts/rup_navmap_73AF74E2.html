Артефакт: Схема навигации
Этот артефакт описывает структуру элементы пользовательского интерфейса системы, а также возможные пути навигации.
Домены: Анализ и проектирование
Типы рабочих продуктов: Модель
Назначение

Для системы существует одна схема навигации. Назначение схемы навигации - выразить основные пути пользовательского интерфейса в системе. Это основные пути через экраны системы, но не обязательно все возможные пути. Ее можно представить как карту дорог пользовательского интерфейса системы.

Схема навигации служит в качестве фона и связи между отдельными Раскадровками.  Раскадровки описывают навигацию перемещений пользователя по элементам пользовательского интерфейса для выполнения системных функций, а схема навигации определяет допустимые пути навигации. Схема навигации выражает структуру пользовательского интерфейса системы, а раскадровка - динамику.

Схема навигации облегчает оценку того, сколько "щелчков" должен сделать пользователь, чтобы достичь определенного экрана.

Опции представления

Для схемы навигации можно использовать различные представления. Ниже приведены некоторые примеры:

  • Иерархическая диаграмма в виде "дерева", каждый уровень которой показывает число щелчков для достижения определенного элемента пользовательского интерфейса
  • Изображение в свободной форме с пользовательскими значками.

Выбранное представление и все решения по настройке следует задокументировать в относящихся к проекту рекомендациях.
