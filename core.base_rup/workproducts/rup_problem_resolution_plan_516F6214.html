Артефакт: План устранения неполадок
Этот артефакт описывает процесс, применяемый для отчета, анализа и устранения неполадок, возникающих в проекте.
Типы рабочих продуктов: План
Назначение

Для описания упорядоченного процесса подготовки отчета, анализа и устранения неполадок , возникающих в проекте.

Взаимосвязи
Артефакт-контейнер
  • План разработки программного обеспечения
РолиОтветственный:
  • Руководитель проекта
Изменен:
  • Руководитель проекта
ЗадачиВход для:
  • Компановка плана разработки программного обеспечения

  • Обработка исключительных ситуаций и неполадок
  • Проверка запросов изменений
  • Разработка плана контроля качества
Выход из:
  • Разработка плана устранения неполадок
Доводка
Опции представления

План устранения неполадок физически может являться частью плана разработки программного обеспечения в случае простой среды проекта. В случае наличия в проекте сложных взаимодействий (обзоры, проверки, оценки и так далее) с многими заинтересованными лицами может быть целесообразно его отделить.
