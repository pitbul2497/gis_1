Артефакт: Рекомендации по проекту
Этот рабочий продукт предоставляет директивное руководство по выполнению определенной задачи или набора задач в контексте проекта.
Типы рабочих продуктов: Процесс
Назначение

Рекомендации по проекту применяются участниками проекта при выполнении присвоенных им задач. Все решения по настройке рабочих продуктов проекта следует фиксировать в рекомендациях по проекту.

Описание
Краткая схема

Структура зависит от того, какие рабочие продукты будут создаваться в проекте.  Обычно существует по крайней мере один раздел для каждого рабочего продукта.

Иллюстрации
Примеры
  • CSPS Creative Design Brief - Inception Phase
  • CSPS Design Comps - Inception Phase
Ключевые условия

Рекомендации обычно выбираются из базового хранилища, управляемого группой процесса в организации. Этот рабочий продукт является временным заменителем для конкретных рекомендаций, выбранных для проекта.

Доводка
Опции представления

Предоставляемые рекомендации зависят от конкретного проекта и основаны на том, что доступно во время выполнения проекта. Рекомендации могут относиться к созданию определенного рабочего продукта либо описывать определенную методику.

Каждую рекомендацию следует настроить в соответствии с конкретным контекстом проекта.
