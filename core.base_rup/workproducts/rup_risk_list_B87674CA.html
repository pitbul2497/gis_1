Артефакт: Список рисков
Этот артефакт представляет собой отсортированный список известных и открытых рисков проекта, отсортированных в порядке уменьшения важности и связанных с определенными действиями по снижению рисков и аварийных ситуаций.
Домены: Управление проектом
Типы рабочих продуктов: Данные проекта

Для  фиксации осознаваемых рисков для успеха проекта.

Описание
Основное описание

В списке рисков  идентифицируются (в порядке уменьшения приоритета) события, которые могут привести к значительным негативным последствиям. Он служит в качестве фокусной точки для деятельностей проекта, и является основой для организации итераций.

Доводка
Опции представления

Список рисков должен фиксировать критические и серьезные риски - если в списке их более 20, то следует тщательно обдумать, действительно ли эти риски являются серьезными. Мониторинг более 20 рисков является тяжелой задачей.
