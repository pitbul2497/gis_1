Артефакт: План управления рисками
Это артефакт подробно описывает управление рисками, связанными с проектом. Подробно описываются задачи управления рисками, которые будут выполняться, присвоенные ответственности и все дополнительные ресурсы, требуемые для деятельности управления рисками. В небольших проектах этот план может быть включен в план разработки программного обеспечения.
Типы рабочих продуктов: План
Назначение

Для гарантии того, что риски проекта правильно идентифицированы, проанализаированы, задокументированы, должным образом отслеживаются и контролируются и проводится работа по их снижению.

Описание
Основное описание

План управления рисками  описывает подход, который будет применяться для идентификации, анализа, назначения приоритетов, мониторинга и снижения рисков. Подробно описываются задачи управления рисками, которые будут выполняться, присвоенные ответственности и все дополнительные ресурсы, требуемые для деятельности управления рисками. В небольших проектах этот план может быть включен в план разработки программного обеспечения.

План управления рисками должен обновляться при изменении рисков или стратегий снижения рисков.
