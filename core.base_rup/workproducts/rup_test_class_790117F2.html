Артефакт: Класс пригодности к тестированию
Этот артефакт представляет собой специализированный класс в модели проектирования, который представляет для данного теста алгоритм, который будет поддерживаться программным обеспечением.
Типы рабочих продуктов: Элементы модели
Назначение

Назначение Класса пригодности к тестированию - фиксация проекта для относящейся к данному тесту функциональности, необходимой для облегчения тестирования. Эта относящаяся к тесту функциональность должна быть включена в модель проектирования программного обеспечения, так чтобы ее можно было разложить в завершенный проект программного обеспечения. Существуют различные относящихся к тесту типы алгоритмов. Два из них приведены ниже:

  • "Заготовки" для классов проектирования, которые необходимо имитировать (обычно потому, что полная версия еще недоступна).
  • Специализированные интерфейсы или вывод, которые предоставляют область видимости или управление, необходимые для проведения тестирования.
