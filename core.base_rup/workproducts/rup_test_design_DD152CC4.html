Описание
Основное описание

Проект тестирования включает все решения, относящиеся к структурным элементам теста и их кооперациям, необходимым для выполнения тестирования уже определенных целей. Этот рабочий продукт приводит в действие деятельности реализации теста и усиливает фокус тестирования во время проектирования и реализации.

Краткая схема

Для каждого целевого элемента тестирования следует определить и описать все механизмы тестирования, а также присвоить все тестовые ответственности соответствующим структурным элементам теста. Для каждого значимого варианта тестирования проект должен задавать реализацию в терминах кооперирования элементов, используя диаграммы кооперирования и/или последовательностей.
