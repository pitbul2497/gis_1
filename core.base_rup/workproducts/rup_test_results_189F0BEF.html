Артефакт: Результаты тестирования
Этот артефакт подводит итоги анализа одного или нескольких протоколов тестирования и запросов изменений, обеспечивая относительно детализированную оценку качества целевых элементов тестирования и состояния тестирования.
Домены: Тестирование

  • Для последующего вычисления различных ключевых показателей тестирования.

Описание
Основное описаниеРезультаты тестирования часто представляют собой большой набор многих результатов тестирования.
Краткая схема

Информация (в противоположность сырым данным), содержащаяся в результатах тестирования, может изменяться в зависимости от технологии и инструментов, которые применялись как во время выполнения теста для фиксации протокола тестирования, так и после этого для проведения анализа сырых данных протокола тестирования.  Ниже перечислены некоторые данные, которые могут быть определены и доступны для проверки и оценки:

  • Идентификатор результатов тестирования (ИД для отличия данных результатов тестирования от других)
  • время, дата, имя ответственного за тестирование и информация о среде (такая как ОС, параметры физической системы и так далее)
  • определенная идентификация целевых элементов тестирования  (версия, объекты и файлы)
  • варианты тестирования, которые должны быть выполнены (и выполнена трассировка к тестируемым требованиям) 
  • выполненные варианты тестирования (и выполнена трассировка к тестируемым требованиям)
  • определение размера целевых элементов тестирования, которое следует выполнить
  • выполненное определение размера целевых элементов тестирования
  • время ответа на заданную последовательность событий
  • данные трассировки, содержащие сведения о диалогах между субъектом и целевыми элементами тестирования и/или между объектами в целевых элементах тестирования.
  • фактический результат каждого выполненного варианта тестирования
  • отличия ожидаемого результата от фактического
  • указание о прохождении или сбое для каждого выполненного варианта тестирования
  • фактический уровень завершенности и положительные результаты для каждого выполненного варианта тестирования
  • все неожиданные или непредусмотренные результаты или алгоритмы
Доводка
Опции представления

При использовании автоматизированных инструментов тестирования, например, из Rational Suite или Test Studio, большая часть определения упомянутой выше информации может быть упрощена с помощью возможностей анализа, имеющихся в этих инструментах. Эти инструменты также предлагают сохраненные и настраиваемые представления информации для отчетов.

Иногда ссылки на результаты тестирования или непосредственно сами результаты включаются в Сводные результаты оценки тестирования.
