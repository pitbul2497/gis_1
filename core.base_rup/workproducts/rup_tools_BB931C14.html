Артефакт: Инструменты
Эти рабочие продукты представляют собой инструменты, необходимые для поддержки разработки программного обеспечения.
Типы рабочих продуктов: Инфраструктура
Назначение

Для процесса разработки программного обеспечения требуются инструменты для поддержки всех деятельностей в жизненном цикле системы.
