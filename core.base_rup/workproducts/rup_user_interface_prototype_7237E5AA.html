Артефакт: Прототип интерфейса
Этот артефакт представляет собой пример пользовательского интерфейса. Он применяется для изучения и/или проверки проекта пользовательского интерфейса.
Домены: Анализ и проектирование
Типы рабочих продуктов: Концепция
Назначение

Прототип пользовательского интерфейса используется следующими ролями:

  • проектировщиками пользовательского интерфейса - для изучения и/или проверки проекта пользовательского интерфейса перед инвестированием в него
  • ответственными за спецификацию требований - для понимания пользовательского интерфейса для варианта использования
  • Аналитиками системы - для понимания влияния пользовательского интерфейса на анализ системы
  • проектировщиками - для понимания влияния пользовательского интерфейса и что для него требуется "изнутри" системы
  • руководителями - для планирования деятельностей разработки и тестирования

Прототипы пользовательских интерфейсов можно использовать для изучения доступного и подходящего проекта пользовательского интерфейса, который удовлетворяет требованиям, что помогает устранить разницу между тем, что требуется (выражено посредством выявления требований), и тем, что осуществимо. Основным назначением создания прототипа пользовательского интерфейса является возможность "тестирования" проекта пользовательского интерфейса, включая удобство работы, до начала фактической разработки . Таким образом, можно убедиться, что будет создана правильная система, до того, как будет затрачено слишком много средств и ресурсов на разработку.

Доводка
Опции представления

Прототипы пользовательского интерфейса могут быть официальными и неофициальными, исполняемыми и неисполняемыми, с высокой и низкой степенью достоверности. Например, прототип пользовательского интерфейса может представлять собой и ряд рисунков, представляющих снимки экрана для нескольких интерактивных HTML-страниц. Формат прототипа UI не имеет особого значения. Важно помнить о назначении прототипа пользовательского интерфейса (для исследования и/или проверки проекта пользовательского интерфейса), и о том, какие навыки требуются для создания прототипа (для создания прототипа пользовательского интерфейса требуются некоторые навыки проектирования пользовательских интерфейсов).

Следует решить, требуется ли прототип для данного проекта.  Решите, какую часть пользовательского интерфейса должен содержать прототип, а также глубину и реализм интерактивных возможностей.  Также решите, следует ли полностью выбросить прототип, либо какие-либо его аспекты планируется включить в конечный продукт.

Помните, что для достижения цели раннего тестирования пользовательского интерфейса разработка прототипа должна быть значительно дешевле разработки реальной системы, в то же время имея достаточно возможностей для поддержки имеющего смысл тестирования его применения.
