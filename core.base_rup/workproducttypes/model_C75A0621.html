Тип рабочего продукта: Модель
Эти рабочие продукты представляют собой специальный тип спецификации, являющейся абстрактным представлением или имитацией "системы", который предоставляет полное описание системы из определенной проекции. Модели часто применяются для достижения лучшего понимания работы системы или для документирования решений по проекту для фактической реализации. Модели часто состоят из нескольких частей различных типов. Эти части называются элементами модели.
