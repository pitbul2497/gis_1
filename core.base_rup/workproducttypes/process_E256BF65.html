Тип рабочего продукта: Процесс
Эти рабочие продукты идентифицируют различные связанные с процессом или управлением артефакты, такие как рекомендации, процедуры, шаблоны, процессы, стандарты и стратегии, которые определяют, каким образом должны быть выполнены (либо предполагается, что будут выполнены) задачи проекта.
Взаимосвязи
Рабочие продукты
  • Прецедент разработки
  • Процесс разработки
  • Рекомендации по проекту
  • Руководство по стилю публикаций
  • Шаблоны проекта
Основное описание

Эти рабочие продукты помогают всем участникам проекта понять, как эффективно выполнять их работу. Они также могут потребоваться для выполнения либо внешних регулирующих законов, либо внутренних правил бизнес-процесса. Некоторые из них являются обязательными и подлежат контролю, другие могут являться просто рекомендациями, например, рекомендации по проекту.
