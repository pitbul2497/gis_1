Тип рабочего продукта: Данные проекта
Эти рабочие продукты идентифицируют информацию, которая применяется для управления проектом. Собранную информацию можно либо хранить в виде постоянных записей, либо использовать исключительно как временную основу в определенной точке в проекте.

Рабочие продукты
  • Атрибуты требований
  • Запрос изменения
  • Показатели проекта
  • Проверка записи
  • Протокол тестирования
  • Рабочее задание
  • Результаты тестирования
  • Список дефектов
  • Список рисков
