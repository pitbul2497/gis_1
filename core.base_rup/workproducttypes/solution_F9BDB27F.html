Основное описание

Многие рабочие продукты вносят вклад в решение или продукт в целом.  Тесты и данные тестов требуются для оценки решения.  Материалы для поддержки пользователей требуются в окончательном продукте.  Исходный код и другие элементы реализации требуются для компоновки окончательного продукта.  Эти рабочие продукты, включая те, которые поставляются вместе с продуктом, относятся к решению.
