Задача: Обработка карт
Назначение
Для корректного отображения необходимой информации на картах требуется ее обработка
Взаимосвязи
РолиОсновной исполнитель:
  • Разрабочик ГИС
Дополнительные исполнители:
  • Лесничий
  • Реководитель проекта
  • Системный администратор
ВходыОбязательный:
  • Разработка ГИС
    		Необязательный:
    • Нет
    Выходы
    • NDVI карта
      Основное описание
      В обработку входит процесс разработки ГИС при помози метожа NDVI
      Шаги
      Выделение изучаемой области
      Фильтрация необходимых спектральных каналов
      Наложение данных в канале на карту
      Визуализация
      Подбор цвета на каждое значение канала
      Ввод требуемых атрибутов в отчет
      Экспорт карт и отчета
