Задача: Сбор и импорт карт MODIS
Назначение

Для реализации ГИС требуется исходный материал в виде специальных спектральных карт

Взаимосвязи
РолиОсновной исполнитель:
  • Разрабочик ГИС
Дополнительные исполнители:
  • Лесничий
  • Реководитель проекта
ВходыОбязательный:
  • Разработка ГИС
    		Необязательный:
    • Нет
    Основное описание

    Процесс подразумевает сбор карт определенного периода (за последние 7 дней) при помощи сервиса LAADS DAAC

    Шаги
    Ввод требуемых данных

    К таким данным относятся: координаты месторасположения ленточного бора, масштаб карты, модель спутника, диапазон даты и времени, необходимые спектральные каналы)

    Ключевые условия

    Правильный ввод координат месторасположения и даты

    Альтернативы

    Импорт карт из системы ГЛОНАС
