Основное описание

Для публикации готовой карты на вер-сайт используется сторонний модуль «qgis2web». Данный плагин в автоматическом режим создает HTML-страницу, взяв за основу все проектные данные созданной ГИС. Для создания такой страницы необходимо настроить ряд параметров, в окне плагина. Технология, применяемая для создания карты, носит название Leaflet.
